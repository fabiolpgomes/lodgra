# 🏠 Home Stay - Sistema de Gestão de Alojamentos

Sistema completo para gestão de propriedades, reservas, despesas e análise financeira.

## ✨ Funcionalidades

- ✅ Gestão de Propriedades
- ✅ Gestão de Reservas (multi-moedas)
- ✅ Sincronização iCal (Airbnb, Booking)
- ✅ Gestão de Despesas
- ✅ Análise Financeira (Lucro Líquido)
- ✅ Calendário Visual
- ✅ Relatórios Exportáveis (Excel)
- ✅ Dashboard com Gráficos
- ✅ Cron Jobs Automáticos

## 🚀 Deploy

Este projeto está configurado para deploy no Vercel.

## 📝 Variáveis de Ambiente

Configure no Vercel Dashboard:

```
NEXT_PUBLIC_SUPABASE_URL=sua-url-do-supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY=sua-chave-anon
CRON_SECRET=sua-chave-secreta-para-crons
```

## 🛠️ Tecnologias
