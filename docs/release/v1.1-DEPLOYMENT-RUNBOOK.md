## Troubleshooting

### Build Fails in Vercel

**Symptom:** Vercel shows "Build failed"

**Steps:**
1. Click on build → View logs
2. Search for error message
3. Common issues:
   - Missing environment variable → Add to Vercel
   - TypeScript error → Fix locally, re-push
   - Dependency issue → Check package.json / package-lock.json

### Production URL Shows 404

**Symptom:** https://homestay.vercel.app shows 404

**Steps:**
1. Check Vercel deployment status (should be ✓ Ready)
2. Clear browser cache (Ctrl+Shift+Delete)
3. Check DNS (if custom domain): `nslookup homestay.example.com`
4. Wait 2-3 min for DNS propagation

### Stripe Webhooks Not Firing

**Symptom:** Checkout completes but no confirmation email / order created

**Steps:**
1. Check Stripe webhook status (should be ✓ Enabled)
2. View webhook logs in Stripe Dashboard
3. Check `/api/stripe/webhook` logs in Vercel
4. Verify `STRIPE_WEBHOOK_SECRET` is correct in Vercel env vars
5. Test webhook delivery: Stripe → [Endpoint] → "Send test event"

### Database Connection Error

**Symptom:** 500 error on dashboard, "connection timeout"

**Steps:**
1. Check Supabase status page
2. Verify `SUPABASE_SERVICE_ROLE_KEY` in Vercel (not expired)
3. Check Supabase logs: SQL Editor → View connections
4. Restart Supabase (if owned): Project Settings → Restart

### Email Not Sending

**Symptom:** Users don't receive confirmation emails

**Steps:**
1. Check Resend logs: View email delivery
2. Verify `RESEND_API_KEY` in Vercel (is valid)
3. Check email template in code: `src/lib/email/`
4. Test Resend API directly: `curl -X POST https://api.resend.com/emails`
