## What's New in v1.1

### 🎫 Public Booking Engine

Direct guests can now book your properties without logging in.

**Features:**
- **Public Property Pages** — Each property has a shareable, public URL
- **Interactive Availability Calendar** — Guests see real-time availability
- **Instant Booking** — Select dates, enter details, pay via Stripe
- **Booking Confirmation** — Email confirmation with QR code & details
- **Multi-Currency** — Book in EUR or BRL

**For Property Managers:**
- View all bookings (direct + imported from Airbnb/Booking.com)
- Filter by channel (Airbnb, Booking, Direct, etc.)
- See occupancy forecasting
- Manage cancellations & rescheduling

**How to Enable:**
1. Go to Property Settings
2. Toggle "Public Booking Enabled"
3. Share the public URL with guests

### 💰 Pricing Tiers & SaaS Model

Home Stay is now a subscription platform with 3 flexible tiers.

**Pricing:**

| Plan | Price | Properties | Features |
|------|-------|-----------|----------|
| **Starter** | €19/month | Up to 3 | Basic calendar, manual bookings |
| **Professional** | €49/month | Up to 10 | Financial reports, fiscal compliance (PT) |
| **Business** | €99/month | Unlimited | All features, priority support, 2FA (coming) |

**What This Means:**
- Organizations select a plan during signup
- Property limit enforced automatically
- Premium features available only on Professional/Business
- Upgrade anytime with pro-rata billing

**Automatic Enforcement:**
- Starter org tries to add 4th property → Error: "Upgrade to continue"
- Starter org tries to access reports → Upgrade prompt with pricing
- Seamless upgrade flow via Stripe checkout

### 📊 Financial Reporting MVP

Real-time insights into property performance.

**New Reports:**

1. **Revenue Per Available Room (RevPAR)**
   - Shows earnings per available night
   - Track occupancy % over time
   - Compare performance month-to-month

2. **Profit & Loss (P&L) Statement**
   - Gross Revenue → Deductions → Net Profit
   - Separate platform fees from operational expenses
   - Color-coded margin analysis (green >30%, yellow 10-30%, red <10%)
   - Export to Excel for accounting

3. **Revenue by Channel**
   - See which platforms generate most revenue
   - Booking.com vs Airbnb vs Direct comparison
   - Identify over-dependence on single channel
   - Risk indicator: Red (>50% from one source)

4. **Cash Flow Forecast**
   - Preview next 30/60/90 days of confirmed bookings
   - Plan ahead for peak seasons
   - Monthly breakdown of expected revenue

**For All Tiers:**
- Historical analysis (lookback period configurable)
- Filter by property or time range
- Export all reports to Excel
- Automatic calculations (no manual entry needed)

## Technical Details

### Stack

- **Frontend:** Next.js 15 (App Router), React 18, TypeScript, Tailwind CSS
- **Backend:** Vercel Functions (serverless), Node.js 18+
- **Database:** Supabase (PostgreSQL), Row-Level Security
- **Payments:** Stripe (checkout + subscriptions)
- **Email:** Resend
- **Hosting:** Vercel

### Architecture Highlights

- Multi-tenancy at database level (org isolation)
- API routes follow RESTful conventions
- Client-side hydration for interactivity
- Cached queries for performance
- Webhooks for async processing (payments, emails)

### Infrastructure

- Auto-scaling (handled by Vercel)
- 99.95% uptime SLA
- Daily automated backups (Supabase)
- CDN-cached static assets
- DDoS protection (Vercel + Cloudflare)
