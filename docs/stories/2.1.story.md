## Story

**Como** proprietário do produto Home Stay,
**Quero** que a landing page esteja optimizada para motores de busca (Google, Bing),
**Para que** o sistema apareça nos primeiros resultados para pesquisas relevantes em Portugal e no Brasil, gerando tráfego orgânico qualificado sem custo de aquisição.

---

## Contexto de Negócio

A landing page actual (`/`) já serve conteúdo bilingue (PT/BR) mas carece de toda a camada de SEO técnico:

- `layout.tsx` tem título e descrição genéricos ("Home Stay - Gestão de Alojamentos")
- Não existem tags Open Graph nem Twitter Card
- Não existe `sitemap.xml` nem `robots.txt`
- Não existe Structured Data (JSON-LD) — oportunidade de rich snippets no Google
- Não existe `hreflang` para segmentação PT-PT vs PT-BR
- Páginas internas (dashboard, admin, reports) não têm `noindex`

O Next.js 15 suporta nativamente `generateMetadata`, `sitemap.ts`, `robots.ts` e `opengraph-image` — implementação sem dependências externas.

### Exemplo de generateMetadata para page.tsx

```typescript
import type { Metadata } from 'next'

export async function generateMetadata(): Promise<Metadata> {
  const appUrl = process.env.NEXT_PUBLIC_APP_URL || 'https://homestay.app'

  return {
    title: 'Home Stay — Gestão de Alojamentos no Airbnb e Booking',
    description: 'Gerencie todos os seus imóveis no Airbnb e Booking num único lugar. Calendário unificado, relatórios financeiros e notificações automáticas. Começa a partir de €9,90/imóvel/mês.',
    keywords: ['gestão alojamentos', 'airbnb booking software', 'PMS curta duração', 'property management system'],
    alternates: {
      canonical: appUrl,
      languages: {
        'pt-PT': appUrl,
        'pt-BR': appUrl,
      },
    },
    openGraph: {
      title: 'Home Stay — Gestão de Alojamentos no Airbnb e Booking',
      description: 'Calendário unificado, relatórios de lucro real e notificações automáticas para anfitriões.',
      url: appUrl,
      siteName: 'Home Stay',
      locale: 'pt_PT',
      alternateLocale: ['pt_BR'],
      type: 'website',
      images: [{ url: `${appUrl}/opengraph-image`, width: 1200, height: 630 }],
    },
    twitter: {
      card: 'summary_large_image',
      title: 'Home Stay — Gestão de Alojamentos',
      description: 'Airbnb + Booking num só lugar. Relatórios reais. A partir de €9,90/imóvel/mês.',
      images: [`${appUrl}/opengraph-image`],
    },
  }
}
```

### Exemplo de sitemap.ts

```typescript
import { MetadataRoute } from 'next'

export default function sitemap(): MetadataRoute.Sitemap {
  const baseUrl = process.env.NEXT_PUBLIC_APP_URL || 'https://homestay.app'
  return [
    { url: baseUrl, lastModified: new Date(), changeFrequency: 'weekly', priority: 1 },
    { url: `${baseUrl}/login`, lastModified: new Date(), changeFrequency: 'monthly', priority: 0.5 },
    { url: `${baseUrl}/register`, lastModified: new Date(), changeFrequency: 'monthly', priority: 0.5 },
  ]
}
```
