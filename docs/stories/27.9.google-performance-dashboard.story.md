# Story 27.9 — Google Vacation Rentals Performance Dashboard

**Epic:** 27 — Google Vacation Rentals Integration  
**Status:** Draft

**Type:** Backend / Dashboard Feature  

---

## 📋 Story

**As a** property manager  
**I want** to view clicks, impressions, and CTR from Google Vacation Rentals  
**So that** I can measure the performance of my properties on Google's platform  

---

## 🎯 Acceptance Criteria

### AC1: Google Merchant Center API Integration
- [ ] Setup connection to Google Merchant Center API (v1)
- [ ] Authenticate using OAuth 2.0 (service account or user account)
- [ ] Implement API client with retry logic (exponential backoff)
- [ ] **Success Metric:** API calls succeed with <5s latency

### AC2: Performance Metrics Sync
- [ ] Fetch daily metrics: impressions, clicks, conversions
- [ ] Fetch weekly and monthly aggregations
- [ ] Store metrics in `google_performance_metrics` table
- [ ] Schedule daily sync at 2 AM UTC (Merchant Center updates ~24h after)
- [ ] **Success Metric:** Metrics updated daily with <2h delay from Google

### AC3: Dashboard Display
- [ ] Show summary card: Total Impressions, Clicks, CTR (%), Conversions
- [ ] Time-range selector: Last 7 days, 30 days, 90 days, custom range
- [ ] Chart: Impressions & Clicks trend (line chart)
- [ ] Chart: CTR trend (line chart with target 2-5%)
- [ ] **Success Metric:** All charts render within 2s

### AC4: Property-Level Breakdown
- [ ] Show performance per property (table with sortable columns)
- [ ] Columns: Property Name, Impressions, Clicks, CTR%, Conversions, ROAS (if bookings linked)
- [ ] Filter by property status (active, archived, draft)
- [ ] **Success Metric:** Table loads <1s for 50+ properties

### AC5: Comparison & Export
- [ ] Compare two time periods side-by-side (delta %, direction indicator)
- [ ] Export metrics to CSV (date range + property selection)
- [ ] Share dashboard link (read-only, expires after 7 days)
- [ ] **Success Metric:** Export <5s for any date range

### AC6: Error Handling
- [ ] Handle API failures gracefully (retry, fallback to cached data, user notification)
- [ ] Display last-sync timestamp + next-sync ETA
- [ ] Alert if sync fails for >24h
- [ ] **Success Metric:** Zero silent data failures
