# Story 36.10: Config Panel (Web)

**Epic:** 36 (Dynamic Pricing + Calendar Redesign)  
**Story ID:** 36.10  
**Status:** Draft  
**Story Points:** 3  
**Priority:** HIGH  

---

## User Story

As a **Host on Web**, I want **to see and edit pricing settings in a right sidebar** so that **I can manage my property's pricing rules without losing calendar context**.

---

## Description

Build right sidebar (web-only) displaying 3-tab config panel (Preços | Descontos | Disponibilidade). Reuses exact tab component from Story 36.2. Mobile version shows as modal (Story 36.2 already handles this).

**Why:** Web layout shows settings in dedicated panel; keeps all workflows visible.

---

## Acceptance Criteria

### Panel Structure

- [ ] Fixed width: ~25% of viewport (right side)
- [ ] Scrollable vertically if content is long
- [ ] 3 tabs: Preços | Descontos | Disponibilidade
- [ ] Tab switching works (same logic as Story 36.2)

### Tab Content

- [ ] Reuse TabPrecos component from Story 36.2
- [ ] Reuse TabDescontos component from Story 36.2
- [ ] Reuse TabDisponibilidade component from Story 36.2

### Data Binding

- [ ] Fetch config for activePropertyId (from sidebar selection)
- [ ] Pre-populate inputs with existing values
- [ ] Save updates to API (Story 36.1 endpoints)
- [ ] Show success/error messages

### Responsive Web

- [ ] Hidden on mobile (≤768px)
- [ ] Visible on web (>768px)
- [ ] Consistent styling with design

### State Management

- [ ] Share activePropertyId with Sidebar (Story 36.8)
- [ ] Update panel when property changes
