# Story 27.12 — Multi-Platform Distribution Dashboard (Aggregated View)

**Epic:** 27 — Google Vacation Rentals Integration  
**Status:** Draft  
**Story ID:** 27.12  
**Created:** 2026-07-17  
**Type:** Backend / Dashboard Feature  

---

## 📋 Story

**As a** property manager  
**I want** to see performance metrics from all OTA platforms (Google, Airbnb, Booking, VRBO, Flatio) in one dashboard  
**So that** I can compare channels and allocate resources to the best-performing platforms  

---

## 🎯 Acceptance Criteria

### AC1: Unified Platform Metrics
- [x] Aggregate impressions, clicks, bookings from all platforms
- [x] Show per-platform breakdown: Google, Airbnb, Booking, VRBO, Flatio
- [x] Normalize metrics for comparison (impressions per day, CTR%, conversion rate)
- [x] Color-code platforms (Google: blue, Airbnb: red, Booking: yellow, etc)
- [x] **Success Metric:** Dashboard loads aggregated data <2s

### AC2: Platform Comparison Charts
- [x] Impressions comparison (bar chart): Which platform gets most visibility?
- [x] Bookings comparison (bar chart): Which platform converts best?
- [x] Revenue comparison (bar chart): Which platform generates most revenue?
- [x] Performance trend (line chart): How each platform performing over time?
- [x] **Success Metric:** All charts interactive and drillable

### AC3: Market Share Analysis
- [x] Calculate platform market share: % of impressions from each platform
- [x] Calculate booking share: % of bookings from each platform
- [x] Calculate revenue share: % of revenue from each platform
- [x] Show shift over time: Is Google gaining share from Booking?
- [x] **Success Metric:** Calculations verified against platform APIs

### AC4: Channel-Specific Insights
- [x] For each platform, show top-3 properties (by bookings, impressions, revenue)
- [x] Show underperforming channels: "VRBO below 5% market share, consider delisting"
- [x] Show platform-specific tips: "Airbnb: Add verification badge", "Booking: Update description"
- [x] **Success Metric:** Insights actionable and specific per platform

### AC5: Data Synchronization
- [x] Sync with each platform's API (Google, Airbnb, Booking, VRBO, Flatio)
- [x] Schedule syncs appropriately: Google (24h), Airbnb (real-time), Booking (hourly), etc
- [x] Handle sync failures gracefully (cached data, retry logic)
- [x] Show last-sync timestamp per platform
- [x] **Success Metric:** Sync success rate >99%

### AC6: Export & Reporting
- [x] Export comparison report: PDF or CSV with all metrics
- [x] Schedule weekly/monthly comparison emails
- [x] Share dashboard link with team/accountant (read-only)
- [x] Add to financial reports: Revenue attribution by platform
- [x] **Success Metric:** Reports generated <10s
