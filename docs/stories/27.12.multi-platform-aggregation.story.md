# Story 27.12 — Multi-Platform Distribution Dashboard (Aggregated View)

**Epic:** 27 — Google Vacation Rentals Integration  
**Status:** Draft  
**Story ID:** 27.12  
**Created:** 2026-07-17  
**Type:** Backend / Dashboard Feature  

---

## 📋 Story

**As a** property manager  
**I want** to see performance metrics from all OTA platforms (Google, Airbnb, Booking, VRBO, Flatio) in one dashboard  
**So that** I can compare channels and allocate resources to the best-performing platforms  

---

## 🎯 Acceptance Criteria

### AC1: Unified Platform Metrics
- [ ] Aggregate impressions, clicks, bookings from all platforms
- [ ] Show per-platform breakdown: Google, Airbnb, Booking, VRBO, Flatio
- [ ] Normalize metrics for comparison (impressions per day, CTR%, conversion rate)
- [ ] Color-code platforms (Google: blue, Airbnb: red, Booking: yellow, etc)
- [ ] **Success Metric:** Dashboard loads aggregated data <2s

### AC2: Platform Comparison Charts
- [ ] Impressions comparison (bar chart): Which platform gets most visibility?
- [ ] Bookings comparison (bar chart): Which platform converts best?
- [ ] Revenue comparison (bar chart): Which platform generates most revenue?
- [ ] Performance trend (line chart): How each platform performing over time?
- [ ] **Success Metric:** All charts interactive and drillable

### AC3: Market Share Analysis
- [ ] Calculate platform market share: % of impressions from each platform
- [ ] Calculate booking share: % of bookings from each platform
- [ ] Calculate revenue share: % of revenue from each platform
- [ ] Show shift over time: Is Google gaining share from Booking?
- [ ] **Success Metric:** Calculations verified against platform APIs

### AC4: Channel-Specific Insights
- [ ] For each platform, show top-3 properties (by bookings, impressions, revenue)
- [ ] Show underperforming channels: "VRBO below 5% market share, consider delisting"
- [ ] Show platform-specific tips: "Airbnb: Add verification badge", "Booking: Update description"
- [ ] **Success Metric:** Insights actionable and specific per platform

### AC5: Data Synchronization
- [ ] Sync with each platform's API (Google, Airbnb, Booking, VRBO, Flatio)
- [ ] Schedule syncs appropriately: Google (24h), Airbnb (real-time), Booking (hourly), etc
- [ ] Handle sync failures gracefully (cached data, retry logic)
- [ ] Show last-sync timestamp per platform
- [ ] **Success Metric:** Sync success rate >99%

### AC6: Export & Reporting
- [ ] Export comparison report: PDF or CSV with all metrics
- [ ] Schedule weekly/monthly comparison emails
- [ ] Share dashboard link with team/accountant (read-only)
- [ ] Add to financial reports: Revenue attribution by platform
- [ ] **Success Metric:** Reports generated <10s

## ⚠️ Risks

- Data inconsistency across platforms (timestamp, timezone issues)
- API rate limits (especially real-time from Airbnb)
- Revenue attribution (how to fairly split booking revenue across channels)
- Data latency (each platform has different sync speed)
