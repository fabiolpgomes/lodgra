## Risks

- **JSON-LD Escaping Issues**: Improper escaping of quotes, special characters, or HTML entities can break JSON-LD parsing or introduce XSS vulnerabilities. Test thoroughly with security scanning.
- **Multi-Language Content Sync**: Ensuring translated content (name, description, amenities) is accurate across PT-BR, ES, EN requires careful QA. Risk of content mismatch.
- **Image Sizing**: OG image dimensions (1200×630px) must be exact. Incorrect aspect ratios may cause preview display issues on social platforms.
- **Rating Integration**: If reviews are added post-launch, AggregateRating must update dynamically. Risk of stale data if caching strategy not updated.
- **Google Rich Results Latency**: Google's Rich Results Test may take 1-2 weeks to reflect schema changes in live search results. Don't expect immediate SERP impact.
- **Browser Caching**: JSON-LD changes may not appear in browser cache. User must hard-refresh or clear cache to see updates.
