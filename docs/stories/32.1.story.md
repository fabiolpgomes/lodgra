# Story 32.1: API identify-org (Staging Phase 2)

## Story
As a customer logging in to Lodgra, I want the system to identify my organization by my email address, so I can see my company name and logo on the login page without additional input.

## Acceptance Criteria

### AC1: API Endpoint Created
- [ ] POST `/api/auth/identify-org` endpoint exists
- [ ] Accepts `{ email: string }` payload
- [ ] Returns `{ orgName: string, orgSlug: string, orgLogoUrl: string | null }`
- [ ] Validates email format before querying database

### AC2: Data Isolation (Security)
- [ ] Query only returns organization data, not user data
- [ ] Uses Supabase admin client (service_role) to bypass RLS
- [ ] Returns empty/null if email doesn't exist (no user enumeration)
- [ ] Email is case-insensitive lookup

### AC3: Rate Limiting
- [ ] Endpoint is rate-limited via Upstash Redis
- [ ] Limit: 10 requests per minute per IP
- [ ] Returns 429 Too Many Requests when exceeded

### AC4: Error Handling
- [ ] Returns 400 for invalid email format
- [ ] Returns 401 for unauthorized attempts
- [ ] Returns 500 with generic message for database errors
- [ ] No stack traces leaked to client

### AC5: Performance
- [ ] Response time < 200ms (p95)
- [ ] Implements query with appropriate database indexes
- [ ] Cache consideration for logo URLs

## Dev Notes

**Database Query:**
- Join `user_profiles` + `organizations` + `organization_branding`
- WHERE `user_profiles.email` (normalized) = input email
- SELECT org `name`, `slug`, branding `logo_url`

## Testing

**Manual Testing:**
```bash
curl -X POST http://localhost:3000/api/auth/identify-org \
  -H "Content-Type: application/json" \
  -d '{"email":"user@algarve-home-stay.com"}'
```

**Expected Response:**
```json
{
  "orgName": "Algarve Home Stay",
  "orgSlug": "algarve-home-stay",
  "orgLogoUrl": "https://images.lodgra.io/logos/algarve-home-stay.png"
}
```

**Edge Cases:**
- Non-existent email → `{ orgName: null, orgSlug: null, orgLogoUrl: null }`
- Rate limit exceeded → 429 with Retry-After header
- Invalid format → 400 with error message
