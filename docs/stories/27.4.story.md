## 📋 Story

**As a** performance engineer  
**I want** to optimize Core Web Vitals to meet Google's ranking criteria  
**So that** lodgra.io improves SEO ranking and user experience  

---

## 🎯 Acceptance Criteria

### AC1: Largest Contentful Paint (LCP) < 2.5s
- [ ] Baseline LCP measured (from Phase 0)
- [ ] Image optimization: lossy compression, modern formats (WebP)
- [ ] Code splitting implemented (lazy load non-critical JS)
- [ ] **Success Metric:** LCP < 2.5s measured via Lighthouse + PageSpeed Insights
- [ ] Server response time < 600ms

### AC2: First Input Delay (FID) < 100ms
- [ ] JavaScript execution time reduced (long tasks broken into <50ms chunks)
- [ ] Unused JavaScript removed or deferred
- [ ] **Success Metric:** FID < 100ms (verified via Lighthouse)
- [ ] Web Workers used if heavy computation needed

### AC3: Cumulative Layout Shift (CLS) < 0.1
- [ ] Image dimensions specified (prevent reflow)
- [ ] Font loading optimized (font-display: swap)
- [ ] Ad/embed space reserved (height/width attributes)
- [ ] **Success Metric:** CLS < 0.1 (verified via Lighthouse)
- [ ] No layout shifts on user interaction (clicks, scrolls)

### AC4: Additional Performance Metrics
- [ ] First Contentful Paint (FCP) < 1.8s
- [ ] Time to Interactive (TTI) < 3.8s
- [ ] **Success Metric:** Lighthouse score > 90 (all pages)
- [ ] PageSpeed Insights: "Good" rating on mobile

### AC5: Performance Monitoring Setup
- [ ] Core Web Vitals events configured in GA4
- [ ] Real User Monitoring (RUM) dashboard created
- [ ] **Success Metric:** CWV metrics tracked daily
- [ ] Alerts configured (if LCP > 3s or CLS > 0.15)

## 📝 Dev Notes

### Image Optimization

- Use next/image for automatic format conversion (WebP)
- Target sizes: <50KB for thumbnails, <200KB for hero images
- Implement srcset for responsive images
- Enable lazy loading (loading="lazy")

### Code Splitting

- Dynamic imports for above-fold vs below-fold components
- Route-based code splitting (Next.js automatic)
- Vendor code split (separate chunks for dependencies)

### Monitoring

- Use web-vitals library for CWV measurement
- Setup GA4 events for LCP, FID, CLS
- Create performance dashboard (Grafana or simple GA4 dashboard)
