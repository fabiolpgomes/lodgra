# Story 15.3: Pull Sync & Painel de Configuração de Canal

## Status
Ready for Review

## Story

**As a** gestor de propriedades,  
**I want** configurar as credenciais do Booking.com no dashboard e sincronizar reservas históricas com um clique,  
**so that** todas as reservas anteriores sejam importadas automaticamente com dados completos.

## Acceptance Criteria

1. Página de configuração de canais acessível em `/[locale]/settings/channels` (ou tab em Settings existente)
2. Utilizador introduz `property_id` externo e `api_key` do Booking.com por propriedade
3. Sistema valida credenciais contra API do Booking.com antes de guardar — exibe erro claro se inválidas
4. Credenciais guardadas de forma segura (nunca expostas no frontend — guardadas como env ou em tabela encriptada)
5. Pull inicial importa reservas dos últimos 90 dias ao activar o canal
6. Indicadores visuais: "Último sync: há X minutos", "X reservas importadas"
7. Botão "Sincronizar agora" dispara pull manual (feedback de loading + resultado)
8. iCal sync existente mantém-se activo para propriedades sem canal API configurado (sem regressão)
9. Feature protegida por variável de ambiente `BOOKING_CHANNEL_ENABLED=true` (off por default)
