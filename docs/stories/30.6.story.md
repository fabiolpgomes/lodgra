# Story 30.6 — Premium Add-on Gate: Feature Flag + Billing

**Epic:** 30 — WhatsApp/SMS Automated Operations  
**Status:** Draft  
**Prioridade:** P2  
**Tipo:** Backend (Billing + Feature Flags)  
**Agente:** @dev  
**Depende de:** 30.1–30.5, Epic 12 (Stripe)  
**Complexity:** LOW (3 story points)

---

## Story

**Como** administrador do produto,  
**Quero** bloquear WhatsApp Automation para planos Essential e disponibilizá-la como add-on,  
**Para que** apenas clientes Expansão/Premium ou assinantes do add-on tenham acesso.

---

## Critérios de Aceitação

- [ ] **AC1** — Feature `whatsapp_automation` activada para planos: `expansao`, `premium`
- [ ] **AC2** — Plano `essential`: acesso bloqueado com paywall + opção de add-on
- [ ] **AC3** — Add-on `whatsapp_automation_addon` disponível como Stripe product separado
- [ ] **AC4** — `organizations` tem campo `whatsapp_automation_enabled: boolean` (default false)
- [ ] **AC5** — Webhook Stripe activa o campo em upgrade para Expansão/Premium
- [ ] **AC6** — Webhook Stripe activa o campo ao assinar add-on
- [ ] **AC7** — Todas as rotas de configuração WhatsApp protegidas por middleware
- [ ] **AC8** — UI de paywall com CTA de upgrade
- [ ] **AC9** — Testes: feature flag, webhook activation, middleware (4+ testes)
