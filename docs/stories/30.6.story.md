## Story

**Como** administrador do produto,  
**Quero** bloquear WhatsApp Automation para planos Essential e disponibilizá-la como add-on,  
**Para que** apenas clientes Expansão/Premium ou assinantes do add-on tenham acesso.

## Implementation Notes

- Feature check considers: plan (premium/expansao) OR add-on flag (whatsapp_automation_enabled)
- Essential plan blocks access by default
- Middleware returns 403 with `upgrade_required: true` flag
- Paywall shows plan comparison with CTA buttons
- Upgrade button → `/settings/billing?tab=upgrade`
- Add-on button → `/settings/billing?tab=addons&addon=whatsapp_automation`
- Integration with Stripe webhooks deferred (can be implemented in separate task)
