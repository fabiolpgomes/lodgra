## Story

**Como** hóspede a visitar a página pública da propriedade,
**Quero** ver o detalhe das casas de banho (tipo + equipamentos),
**Para que** saiba exactamente o que a propriedade oferece sem ter de perguntar.

---

## Context

Story 21.1 adicionou quartos com tipos de cama à página pública. O mesmo padrão deve ser
aplicado às casas de banho (`property_bathrooms`), que já têm editor no painel admin (Epic 18.4)
mas não eram visíveis na página pública. A secção aparece entre Quartos e Políticas.

## Dev Notes

### Query (p/[slug]/page.tsx)
```ts
const { data: propertyBathroomsRaw } = await adminClient
  .from('property_bathrooms')
  .select('id, name, bathroom_type, amenities')
  .eq('property_id', property.id)
  .order('sort_order')
```

Usa o mesmo padrão de adminClient que os quartos (contorna RLS para página pública).
