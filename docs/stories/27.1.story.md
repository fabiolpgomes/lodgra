## 📝 Dev Notes

### Tools Required

**Free/Low-Cost Tools:**
- **Google Search Console** (free) — Domain verification, indexation status, search performance
- **Google Analytics 4** (free) — Event tracking, conversion tracking, traffic analysis
- **Google Keyword Planner** (free) — Search volume + CPC estimates (requires Google Ads account)
- **Screaming Frog** (free tier) — Website crawl, technical audit, broken links
- **Google PageSpeed Insights** (free) — Core Web Vitals measurement
- **Google Mobile-Friendly Test** (free) — Mobile compatibility check

**Optional Paid Tools (if budget allows):**
- **Ahrefs** ($99+/month) — Keyword research, competitor backlinks, traffic estimation
- **SEMrush** ($119+/month) — Comprehensive SEO tool
- **Ubersuggest** ($12/month) — Keyword research alternative

**Recommendation:** Start with free tools (Google suite + Screaming Frog). If keyword volume insufficient, consider 7-day Ahrefs trial.
