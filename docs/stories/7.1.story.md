## Story

**Como** gestor de propriedades,
**Quero** exportar o relatório do proprietário em formato Excel (XLSX),
**Para que** eu possa partilhar um ficheiro editável com o proprietário ou com o contabilista.

---

## Contexto de Negócio

A Story 6.2 implementou o relatório do proprietário com export via `window.print()` (PDF via browser). O PDF é útil para envio rápido, mas property managers e contabilistas frequentemente precisam de um ficheiro Excel para análise e customização posterior.

Esta story adiciona export XLSX à página `/owners/[id]/report` reutilizando o padrão já existente em `PropertyAnalysis.tsx` (que usa `xlsx` ou similar).

### Implementação Esperada

```tsx
// src/app/owners/[id]/report/page.tsx
import * as XLSX from 'xlsx'  // ou a lib que já está a ser usada

function exportToExcel(owner, properties, summary, period) {
  const rows = properties.map(p => ({
    'Propriedade': p.name,
    'Receita Bruta': p.revenue,
    'Comissão (%)': p.management_percentage,
    'Comissão €': p.managementFee,
    'Despesas': p.expenses,
    'Receita Líquida': p.ownerNet,
  }))

  rows.push({
    'Propriedade': 'TOTAL',
    'Receita Bruta': summary.revenue,
    'Comissão (%)': '',
    'Comissão €': summary.managementFee,
    'Despesas': summary.expenses,
    'Receita Líquida': summary.ownerNet,
  })

  const ws = XLSX.utils.json_to_sheet(rows)
  const wb = XLSX.utils.book_new()
  XLSX.utils.book_append_sheet(wb, ws, 'Relatório')
  XLSX.writeFile(wb, `relatorio-${owner.full_name}-${period}.xlsx`)
}
```
