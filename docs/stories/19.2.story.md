## Story

**Como** operador do Lodgra,
**Quero** um endpoint público `/api/google-feed` com dados estruturados de todas as propriedades activas,
**Para que** possa submeter o feed ao Google Hotel Center e aparecer nos resultados de pesquisa Google.

---

## Acceptance Criteria

- [x] **AC1** — `GET /api/google-feed` retorna JSON com todas as propriedades `is_public=true`
- [x] **AC2** — Cada propriedade inclui: nome, descrição, tipo, endereço, fotos, hóspedes, quartos, casas de banho
- [x] **AC3** — Cada propriedade inclui dados de preço: base_price, currency, min_nights, cleaning_fee, pet_fee
- [x] **AC4** — Cada propriedade inclui políticas: checkin_from, checkin_until, checkout_until
- [x] **AC5** — Cada propriedade inclui disponibilidade: datas bloqueadas (reservas confirmed/pending nos próximos 365 dias)
- [x] **AC6** — Cada propriedade inclui amenidades (da tabela property_amenities)
- [x] **AC7** — Headers: `Cache-Control: public, max-age=3600, s-maxage=3600` (cache 1h CDN)
- [x] **AC8** — Endpoint público (sem autenticação) — usa `createAdminClient()` para bypass RLS
- [x] **AC9** — `generatePropertyJsonLd` actualizado para usar checkin/checkout reais em vez de hardcoded
- [x] **AC10** — Build PASS + Lint PASS
