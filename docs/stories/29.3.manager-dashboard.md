## Story

**Como** manager,  
**Quero** visualizar, criar e atribuir tarefas de limpeza,  
**Para que** possa coordenar o trabalho dos cleaners.

---

## Acceptance Criteria (Granular)

### AC1: Page `/[locale]/cleaning/manage` renderiza tabela de tasks
- [ ] **GIVEN** manager logged in
- **WHEN** navega para `/pt-BR/cleaning/manage`
- **THEN** página carrega em < 2s
- [ ] **AND** tabela mostra todas as tasks da organização

### AC2: Tabela tem colunas corretas
- [ ] **WHEN** tabela renderiza
- **THEN** colunas visíveis: Property, Scheduled Date, Cleaner, Status, Actions
- [ ] **AND** status badge colourida (pending=yellow, in_progress=blue, done=green, issue=red)

### AC3: Filtros funcionam (property, date range, status, cleaner)
- [ ] **GIVEN** tabela com 20+ tasks
- **WHEN** filtra por property_id
- **THEN** tabela mostra apenas tasks daquela property
- [ ] **AND** filtro múltiplo funciona (property + status)

### AC4: Ações na tabela (View, Edit, Assign, Mark Done, Delete)
- [ ] **GIVEN** row na tabela
- **WHEN** clica "View Detail"
- **THEN** navega para `/cleaning/tasks/[taskId]`
- [ ] **AND** "Edit" abre form inline ou modal
- [ ] **AND** "Assign Cleaner" abre select com lista de cleaners
- [ ] **AND** "Mark Done" muda status para done (popup confirm)
- [ ] **AND** "Delete" com confirmação

### AC5: Form criar nova task (create button)
- [ ] **GIVEN** manager na página manage
- **WHEN** clica botão "+ Nova Tarefa"
- **THEN** abre form com campos:
  - property_id (required, select)
  - scheduled_date (required, datepicker)
  - scheduled_time (optional, timepicker)
  - cleaner_id (optional, select)
  - checklist_template_id (optional, select)
  - notes (optional, textarea)

### AC6: Form é responsivo (mobile)
- [ ] **GIVEN** form em viewport mobile (375px)
- **WHEN** renderiza
- **THEN** campos stack verticalmente
- [ ] **AND** buttons são touch-friendly (48px+ height)
- [ ] **AND** select dropdowns scrollam (não overflow)

### AC7: Validações no form
- [ ] **WHEN** submite form sem property_id
- **THEN** erro: "Propriedade é obrigatória"
- [ ] **AND** sem scheduled_date: "Data é obrigatória"
- [ ] **AND** scheduled_date < hoje: "Data não pode ser no passado"

### AC8: Form submit POST `/api/cleaning`
- [ ] **GIVEN** form válido
- **WHEN** clica "Salvar"
- **THEN** POST to `/api/cleaning` com payload
- [ ] **AND** response 201 retorna task criada
- [ ] **AND** tabela atualiza sem reload (optimistic update)

### AC9: Integração com design-system Lodgra
- [ ] **WHEN** página renderiza
- **THEN** usa buttons do shadcn (primary, secondary, danger)
- [ ] **AND** usa Table component Lodgra
- [ ] **AND** usa Form component (react-hook-form)
- [ ] **AND** dark mode funciona (ThemeProvider)

### AC10: i18n labels (pt-BR, es-ES, en-US)
- [ ] **GIVEN** locale=es
- **WHEN** página renderiza
- **THEN** "Nueva Tarea" (não "Nova Tarefa")
- [ ] **AND** "Propriedad" (não "Propriedade")
- [ ] **AND** placeholders também traduzidos

### AC11: RLS verificado (manager vê apenas sua org)
- [ ] **GIVEN** manager de org-1
- **WHEN** fetch de `/api/cleaning`
- **THEN** retorna apenas tasks com organization_id = org-1
- [ ] **AND** manager de org-2 não consegue ver

### AC12: Pagination na tabela
- [ ] **GIVEN** > 50 tasks
- **WHEN** tabela renderiza
- **THEN** mostra 20 por página + "Load More" ou pagination controls
- [ ] **AND** URL atualiza com `?page=2`

## Integration Verification (IV)

### IV1: Design-system integration
- [ ] **WHEN** página renderiza
- **THEN** usa mesmos buttons, tables, forms existentes
- [ ] **AND** TypeScript types match (organization_id, etc.)

### IV2: Carrega properties/cleaners de DB
- [ ] **WHEN** form abre
- **THEN** select "Property" carrega de `properties` table
- [ ] **AND** select "Cleaner" carrega de `user_profiles` com guest_type='cleaner'

### IV3: RBAC verificado
- [ ] **WHEN** staff user tenta acessar `/cleaning/manage`
- **THEN** redirects para `/cleaning` (apenas manager)
- [ ] **AND** API `/api/cleaning` rejeita staff (403)

### IV4: i18n consistency
- [ ] **WHEN** muda locale via next-intl
- **THEN** página atualiza sem reload
- [ ] **AND** form labels, buttons, placeholders todos traduzidos
