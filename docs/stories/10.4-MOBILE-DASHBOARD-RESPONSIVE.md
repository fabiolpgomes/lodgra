## Story

As a mobile user, I want the dashboard to be responsive on all screen sizes (320px-1920px+) so that I can view my metrics and charts effectively on any device without horizontal scrolling or broken layouts.

---

## Acceptance Criteria

### AC1: Responsive Card Grid
- [ ] Cards stack vertically on mobile (320px-640px)
- [ ] Cards display in 2-column grid on tablet (640px-768px)
- [ ] Cards display in 3-column grid on desktop (768px+)
- [ ] Padding responsive: `p-4` mobile → `p-6` tablet → `p-8` desktop
- [ ] No horizontal scrolling on any device
- [ ] Grid gap responsive: `gap-4` mobile → `gap-6` tablet → `gap-8` desktop

### AC2: Responsive Charts
- [ ] StatusChart height: `h-48` mobile → `h-64` tablet → `h-80` desktop
- [ ] OccupancyChart height: `h-48` mobile → `h-64` tablet → `h-80` desktop
- [ ] RevenueChart height: `h-48` mobile → `h-64` tablet → `h-80` desktop
- [ ] Charts maintain aspect ratio on all screen sizes
- [ ] Legend positions correctly on mobile (below chart)
- [ ] No chart overflow on small screens

### AC3: Responsive Typography
- [ ] Page title: `text-xl sm:text-2xl lg:text-3xl`
- [ ] Card titles: `text-lg sm:text-xl`
- [ ] Body text: `text-sm sm:text-base`
- [ ] All fonts readable on 320px screens
- [ ] No text truncation on mobile unless necessary

### AC4: Responsive Layout Container
- [ ] Dashboard wrapper: `max-w-7xl mx-auto px-4 sm:px-6 lg:px-8`
- [ ] Safe area padding for notches (iOS)
- [ ] Proper spacing from Header on all devices
- [ ] Bottom nav spacing preserved (56px)

### AC5: Lighthouse Mobile Score
- [ ] Mobile score: 85%+ (from 35%)
- [ ] No mobile usability issues
- [ ] All text readable (16px+)
- [ ] Touch targets 44px+ (icon buttons updated)

**Key Mobile-First Principles:**
1. Base styles apply to mobile (320px)
2. Use `sm:`, `md:`, `lg:` prefixes for larger breakpoints
3. Touch targets minimum 44px (WCAG AA)
4. Responsive padding: `px-4` base → `px-6` tablet → `px-8` desktop
5. Responsive typography: scale up for larger screens

**Breakpoints:**
- `base` (no prefix): 320px - mobile
- `sm:`: 640px - portrait tablet
- `md:`: 768px - landscape tablet
- `lg:`: 1024px - desktop
- `xl:`: 1280px - desktop large
