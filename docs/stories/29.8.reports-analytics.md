---

## Acceptance Criteria

### AC1: Page `/[locale]/cleaning/manage/reports` com filtros
- [ ] **GIVEN** manager authenticated
- **WHEN** acessa `/pt-BR/cleaning/manage/reports`
- **THEN** página carrega com filtros à esquerda

### AC2: Filtros (date range, property, cleaner, status)
- [ ] **GIVEN** formulário de filtros
- **WHEN** seleciona date range (date picker)
- **THEN** relatório atualiza
- [ ] **AND** filtro por property, cleaner, status funcionam (multi-select)

### AC3: Métricas exibidas (total tasks, completion rate, avg time)
- [ ] **WHEN** relatório renderiza
- **THEN** mostra cards com:
  - "Total Tasks: 45"
  - "Completion Rate: 92%"
  - "Avg Time per Task: 1h 15m"
  - "Tasks Today: 12"

### AC4: Tabela com dados granulares
- [ ] **WHEN** tabela renderiza
- **THEN** colunas: Property, Cleaner, Date, Status, Items Completed, Time Spent
- [ ] **AND** dados correspondem a filtros selecionados

### AC5: Chart (bar chart tasks por dia)
- [ ] **WHEN** página renderiza
- **THEN** chart mostra tasks concluídas por dia (últimos 30 dias)
- [ ] **AND** X-axis = datas, Y-axis = número tasks

### AC6: Export CSV com dados
- [ ] **WHEN** clica "Export CSV"
- **THEN** download arquivo: `cleaning-report-2026-05-21.csv`
- [ ] **AND** CSV contém: data, property, cleaner, status, duration

### AC7: Mobile-friendly
- [ ] **GIVEN** viewport mobile
- **WHEN** relatório renderiza
- **THEN** tabela scrollable horizontally
- [ ] **AND** charts responsivos

### AC8: Performance (carrega em < 2s)
- [ ] **WHEN** aplica filtros
- **THEN** dados recarregam em < 2s
- [ ] **AND** queries otimizadas com índices

### AC9: Nenhum dado de outra org é visível
- [ ] **GIVEN** manager de org-1
- **WHEN** relatório carrega
- **THEN** apenas tasks de org-1 são exibidos (RLS)

## Testing

**E2E:**
```typescript
// Apply filters
await page.goto('/pt-BR/cleaning/manage/reports');
await page.fill('input[name="date-from"]', '2026-05-01');
await page.fill('input[name="date-to"]', '2026-05-21');
await page.selectOption('[name="property"]', 'prop-123');
await page.click('button:has-text("Apply Filters")');
await expect(page.locator('text=Completion Rate')).toBeVisible();

// Export
await page.click('button:has-text("Export CSV")');
const downloadPromise = page.waitForEvent('download');
const download = await downloadPromise;
expect(download.suggestedFilename()).toContain('cleaning-report');
```

## Dev Notes

- Use Recharts para charts (Chart.js também ok)
- API agregação: `SELECT property_id, COUNT(*) as total_tasks, SUM(CASE WHEN status='done' THEN 1 ELSE 0 END) as completed_tasks FROM cleaning_tasks GROUP BY property_id`
- CSV export via Papa Parse ou manual string building
