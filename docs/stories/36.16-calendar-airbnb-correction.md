## ⚠️ CONTEXTO CRÍTICO

**Problema:** Stories 36.14 (Mobile) e 36.15 (Desktop) foram implementadas e aprovadas, MAS não correspondem ao padrão Airbnb visual solicitado.

**Descoberta:** Ao testar em produção, nenhuma das versões (mobile ou desktop) reflete o layout Airbnb que foi enviado como referência.

**Causa Raiz:** Falha de validação visual durante dev + QA. Faltou comparação explícita contra screenshots de referência.

**Ação:** Story 36.16 é RETRABALHO para corrigir 36.14 + 36.15 e entregar o padrão Airbnb correto.

---

## User Story

As a **Host**, I want **to manage my calendar and properties in a clean, intuitive Airbnb-style interface** so that **I can easily see availability, prices, and settings without confusion** (desktop + mobile).

---

## Description

**Story 36.14 e 36.15 falharam em entregar o padrão Airbnb.**

Este retrabalho implementará corretamente:

### Mobile Layout
- **Property List View:** Cards horizontais com imagem + nome + tipo + localização + dots de disponibilidade
- **Calendar View:** Grid 7x7 com preços por dia, indicador "hoje" (círculo vermelho), reservas com nomes
- **Settings Tabs:** Preços | Descontos | Disponibilidade com cards grandes e legíveis

### Desktop Layout  
- **Left Sidebar:** Lista de propriedades com mini calendário
- **Center:** Calendário detalhado (mesmo grid que mobile)
- **Right Sidebar:** Settings com tabs e cards

### Referência Visual Obrigatória
Use **exatamente** as imagens Airbnb enviadas:
- Mobile property list (horizontal cards com dots)
- Mobile calendar (grid com preços)
- Mobile settings (tabs + cards grandes)
- Desktop layout (sidebar + calendar + sidebar)

---

## Scope

**IN (OBRIGATÓRIO - Retrabalho):**
- Mobile: Cards de propriedades (horizontal, imagem 80px, nome, tipo, localização, 15 dots)
- Mobile: Grid calendário (7 colunas, preços grandes, hoje em vermelho, nomes dos hóspedes)
- Mobile: Tabs de configurações (Preços | Descontos | Disponibilidade) com cards grandes
- Desktop: Sidebar esquerda (280px) com lista de propriedades
- Desktop: Centro com calendário (mesmo grid mobile)
- Desktop: Sidebar direita (300px) com settings
- Responsive: 375px (mobile) → 768px (tablet) → 1024px (desktop)
- Visual: **DEVE PARECER COM AIRBNB**, não com Lodgra antigo

**OUT (Não incluído nesta story):**
- Drag-and-drop
- Deep linking/URL routing
- Analytics
- Notificações

## Acceptance Criteria

### AC Visuais (CRÍTICA — Comparar com Imagens Airbnb)

#### Mobile - Property List (Screenshot referência)
- [ ] Cards dispostos verticalmente (não horizontalmente como antes)
- [ ] Cada card tem:
  - [ ] Imagem da propriedade (80px altura, 100% width, object-fit cover)
  - [ ] Nome (16px, bold, #2c3e50)
  - [ ] Tipo + Localização (12px, gray, sublinha suave)
  - [ ] Dots de disponibilidade (15 dots em grid 3x5, verde=livre, cinza=booked)
- [ ] Cards com espaçamento 12px gap
- [ ] Padding 16px ao redor da lista

#### Mobile - Calendar Grid (Screenshot referência)
- [ ] Grid 7 colunas (S T Q Q S S D) com headers
- [ ] Cada dia tem:
  - [ ] Número do dia (20px, bold, #2c3e50)
  - [ ] Preço em EUR (14px, #2c3e50)
  - [ ] Se hoje: círculo vermelho (#e74c3c) no número
  - [ ] Se reservado: fundo cinza (#ecf0f1) + nome do hóspede (11px)
- [ ] Altura mínima 100px por célula
- [ ] Espaçamento 4px entre células
- [ ] Mês/ano centralizado no topo (18px, bold)

#### Mobile - Settings Tabs (Screenshot referência)
- [ ] 3 tabs: Preços | Descontos | Disponibilidade
- [ ] Tab ativo com fundo preto (#2c3e50), text white
- [ ] Tab inativo com fundo light gray, text gray
- [ ] Cards dentro das tabs:
  - [ ] Altura mínima 64px-80px
  - [ ] Padding 16px
  - [ ] Título (16px, bold), conteúdo (14px)
  - [ ] Ação buttons/toggles à direita
  - [ ] Espaçamento 12px entre cards

#### Desktop - Layout (Screenshot referência)
- [ ] 3-column grid: 280px | 1fr | 300px
- [ ] Left sidebar: Lista de propriedades (mesmo visual que mobile cards, mas vertical)
- [ ] Center: Calendário grid (idêntico ao mobile)
- [ ] Right sidebar: Settings com tabs (idêntico ao mobile)
- [ ] Todo o layout responsivo e alinhado
