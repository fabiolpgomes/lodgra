## Story

**Como** cleaner,  
**Quero** ver minhas tarefas de limpeza,  
**Para que** saiba o que devo fazer hoje.

---

## Acceptance Criteria

### AC1: Dashboard mostra cards "Today", "Upcoming", "Completed"
- [ ] **GIVEN** cleaner com 3+ tasks
- **WHEN** acessa `/[locale]/cleaning/dashboard`
- **THEN** exibe 3 seções: "Tarefas de Hoje", "Próximas", "Completadas"

### AC2: Cada card mostra property name, date/time, status badge
- [ ] **WHEN** card renderiza
- **THEN** mostra: Property Name, "25 Maio 10:00", Status Badge
- [ ] **AND** status badges colouridas

### AC3: CTA "Começar Limpeza" navega para task detail
- [ ] **WHEN** clica botão em card
- **THEN** navega para `/[locale]/cleaning/tasks/[taskId]`

### AC4: Mobile-friendly (vertical stack)
- [ ] **WHEN** viewport mobile
- **THEN** cards stack vertically (não lado-a-lado)

### AC5: Carrega apenas tasks do cleaner actual
- [ ] **GIVEN** cleaner-A e cleaner-B
- **WHEN** cleaner-A acessa dashboard
- **THEN** vê apenas suas tasks (RLS)

### AC6: Auto-refresh a cada 30s
- [ ] **WHEN** dashboard carrega
- **THEN** refetch tasks a cada 30s (em background)
- [ ] **AND** não pisca/reload (background update)

## Integration Verification

### IV1: Usa AuthLayout existente
- [ ] **WHEN** página renderiza

## Testing

**E2E Tests:**
```typescript
// Cleaner sees only their tasks
await page.goto('/pt-BR/cleaning/dashboard');
const taskCards = await page.locator('[data-testid="task-card"]');
expect(taskCards).toHaveCount(3);

// Auto-refresh
await page.waitForTimeout(31000);
const newCards = await page.locator('[data-testid="task-card"]');
expect(newCards).toHaveCount(4); // New task appeared
```
