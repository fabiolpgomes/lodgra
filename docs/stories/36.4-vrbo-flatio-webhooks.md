### Subtask 2: VRBO/Expedia Webhook
**Files**: `src/app/api/webhooks/vrbo/reservation/route.ts` (novo)

**Estrutura** (seguir padrão de Story 36.3):
```typescript
export async function POST(request: NextRequest) {
  const event = await request.json()
  const signature = request.headers.get('x-vrbo-signature')
  
  // Usar WebhookManager:
  const manager = new WebhookManager()
  if (!await manager.validateSignature('vrbo', event, signature)) {
    return NextResponse.json({ error: 'Invalid' }, { status: 403 })
  }
  
  await manager.updateReservationFromWebhook('vrbo', event.reservation.id, event)
  return NextResponse.json({ success: true })
}
```

**Eventos VRBO**:
- RESERVATION_CREATE → "confirmed"
- RESERVATION_CANCEL → "cancelled"
- RESERVATION_MODIFY → update fields

export function mapVrboEventToUpdate(event: any) {
  return {
    status: vrboStatusMap[event.eventType],
    check_in: event.reservation.checkInDate,
    check_out: event.reservation.checkOutDate,
    number_of_guests: event.reservation.numberOfGuests,
    updated_at: new Date().toISOString()
  }
}

// Flatio (novo)
