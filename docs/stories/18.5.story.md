## Story

**Como** gestor de propriedades,
**Quero** configurar taxas adicionais (limpeza, animais) e horários de check-in/out,
**Para que** hóspedes e sistemas de reserva tenham esta informação correcta.

---

## Acceptance Criteria

- [x] **AC1** — Campo "Taxa de Limpeza": valor numérico + tipo (Por Estadia / Por Noite)
- [x] **AC2** — Campo "Taxa de Animais de Estimação": valor numérico + tipo (Por Estadia / Por Noite)
- [x] **AC3** — Taxa de 0 ou vazio = sem taxa (não obrigatório)
- [x] **AC4** — "Check-in a partir das": campo de hora (HH:MM)
- [x] **AC5** — "Check-in até às": campo de hora (HH:MM, opcional)
- [x] **AC6** — "Checkout até às": campo de hora (HH:MM)
- [x] **AC7** — Dados persistidos nas colunas adicionadas em 18.1
- [x] **AC8** — Valores visíveis no perfil da propriedade
