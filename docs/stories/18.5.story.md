# Story 18.5: Taxas adicionais e horários de check-in/out

**Epic:** 18 — Propriedades: Comodidades, Quartos, Banheiros e Taxas
**Status:** Ready for Dev
**Prioridade:** P2 — Depende de 18.1
**Tipo:** Full Stack
**Agente:** @dev

---

## Story

**Como** gestor de propriedades,
**Quero** configurar taxas adicionais (limpeza, animais) e horários de check-in/out,
**Para que** hóspedes e sistemas de reserva tenham esta informação correcta.

---

## Acceptance Criteria

- [ ] **AC1** — Campo "Taxa de Limpeza": valor numérico + tipo (Por Estadia / Por Noite)
- [ ] **AC2** — Campo "Taxa de Animais de Estimação": valor numérico + tipo (Por Estadia / Por Noite)
- [ ] **AC3** — Taxa de 0 ou vazio = sem taxa (não obrigatório)
- [ ] **AC4** — "Check-in a partir das": campo de hora (HH:MM)
- [ ] **AC5** — "Check-in até às": campo de hora (HH:MM, opcional)
- [ ] **AC6** — "Checkout até às": campo de hora (HH:MM)
- [ ] **AC7** — Dados persistidos nas colunas adicionadas em 18.1
- [ ] **AC8** — Valores visíveis no perfil da propriedade

### Nota estratégica

Horários de check-in/out e taxas são campos obrigatórios no schema Google Vacation Rentals. Implementar com os nomes de coluna correctos (já definidos em 18.1) evita renaming futuro.
