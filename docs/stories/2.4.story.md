## Contexto de Negócio

A landing page (2.1) tem JSON-LD básico (SoftwareApplication). As páginas de propriedade e outros recursos ainda precisam de estrutura completa:

**Oportunidades de rich snippets:**
- **LocalBusiness** — Propriedades (endereço, telefone, rating)
- **Product** ou **Accommodation** — Propriedades com preço, availability
- **BreadcrumbList** — Navegação em Docs
- **FAQSchema** — FAQ pages (futura)
- **Article** — Blog posts (futura)

**Impacto:**
- Rich snippets aumentam CTR em 20-30%
- Google prefere sites com markup validado
- Melhora confiança do utilizador (ratings visíveis)

## Dev Notes

### 1. LocalBusiness para Propriedades

Em `src/app/p/[slug]/page.tsx`:

```typescript
export async function generateMetadata({ params }: { params: { slug: string } }) {
  const property = await getPropertyBySlug(params.slug)

  const jsonLd = {
    '@context': 'https://schema.org',
    '@type': 'LocalBusiness',
    '@id': `${baseUrl}/p/${params.slug}`,
    name: property.name,
    description: property.description,
    address: {
      '@type': 'PostalAddress',
      streetAddress: property.street,
      addressLocality: property.city,
      postalCode: property.postal_code,
      addressCountry: property.country_code.toUpperCase(), // 'PT', 'BR', etc.
    },
    telephone: property.phone || '',
    url: `${baseUrl}/p/${params.slug}`,
    image: {
      '@type': 'ImageObject',
      url: property.photo_url,
      width: 800,
      height: 600,
    },
    aggregateRating: property.rating ? {
      '@type': 'AggregateRating',
      ratingValue: property.rating,
      ratingCount: property.review_count || 0,
    } : undefined,
    geo: {
      '@type': 'GeoCoordinates',
      latitude: property.latitude,
      longitude: property.longitude,
    },
    offers: {
      '@type': 'Offer',
      priceCurrency: property.currency || 'EUR',
      price: property.price_per_night,
      availability: 'https://schema.org/InStock', // ou OutOfStock baseado em reservas
      description: `${property.price_per_night} ${property.currency} por noite`,
    },
    priceRange: property.price_range || '$$', // $, $$, $$$, $$$$
    amenities: property.amenities?.split(',').map(a => ({
      '@type': 'Thing',
      name: a.trim(),
    })) || [],
  }

  return {
    title: property.name,
    description: property.description,
    openGraph: { /* ... */ },
    jsonLd: [jsonLd],
  }
}
```

### 2. BreadcrumbList para Docs

Em `src/app/docs/page.tsx` ou layout:

```typescript
export async function generateMetadata() {
  const breadcrumbs = [
    { position: 1, name: 'Home', url: baseUrl },
    { position: 2, name: 'Documentação', url: `${baseUrl}/docs` },
  ]

  const jsonLd = {
    '@context': 'https://schema.org',
    '@type': 'BreadcrumbList',
    itemListElement: breadcrumbs.map(b => ({
      '@type': 'ListItem',
      position: b.position,
      name: b.name,
      item: b.url,
    })),
  }

  return {
    jsonLd: [jsonLd],
  }
}
```
