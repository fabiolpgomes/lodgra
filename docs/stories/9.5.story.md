## Story

**Como** sistema,
**Quero** confirmar automaticamente a reserva após pagamento Stripe e notificar hóspede e gestor,
**Para que** o ciclo de reserva directa fique completo sem intervenção manual.

---

## Contexto de Negócio

Sem confirmação automática, o gestor teria de confirmar manualmente cada reserva — inviável. Este story fecha o ciclo: pagamento → confirmação → email ao hóspede → email ao gestor → datas bloqueadas no iCal.

O webhook Stripe já existe para subscriptions (`/api/stripe/webhook`). Será estendido para lidar com eventos de reservas directas, ou criado um endpoint separado para clareza.

---

## Critérios de Aceitação

- [ ] **AC1** — Webhook `POST /api/stripe/booking-webhook` processa evento `checkout.session.completed`
- [ ] **AC2** — Após webhook: reserva actualizada para `status = 'confirmed'`
- [ ] **AC3** — Email enviado ao hóspede (via Resend): confirmação com datas, propriedade, valor pago, instruções de contacto
- [ ] **AC4** — Email enviado ao gestor (EMAIL_ADMIN): nova reserva directa com dados do hóspede e datas

- [ ] **AC8** — Reservas directas confirmadas são incluídas no feed iCal export da propriedade (Story 9.2 calendário e iCal export existente)

### Lógica do webhook
```typescript
// Eventos a processar:
// - checkout.session.completed → confirmar reserva
// - checkout.session.expired → cancelar reserva pending_payment

const event = stripe.webhooks.constructEvent(body, sig, process.env.STRIPE_BOOKING_WEBHOOK_SECRET!)

if (event.type === 'checkout.session.completed') {
  const session = event.data.object
  const reservationId = session.metadata.reservation_id

  // Idempotency check
  const existing = await supabase.from('reservations').select('status').eq('id', reservationId).single()
  if (existing.data?.status === 'confirmed') return // Already processed

  // Update reservation
  await supabase.from('reservations').update({
    status: 'confirmed',
    stripe_checkout_session_id: session.id,
    stripe_payment_intent_id: session.payment_intent,
  }).eq('id', reservationId)

  // Send emails
  await sendBookingConfirmationToGuest(reservationId)
  await sendBookingNotificationToManager(reservationId)
}
```
