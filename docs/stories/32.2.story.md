## Testing

**Manual Testing Checklist:**
```bash
# Verify Vercel staging deployment
curl https://staging-lodgra.vercel.app

# Test identify-org endpoint in staging
curl -X POST https://staging-lodgra.vercel.app/api/auth/identify-org \
  -H "Content-Type: application/json" \
  -d '{"email":"staging-test@lodgra.io"}'

# Expected response:
# { "orgName": "Staging Test Org", "orgSlug": "staging-test-org", "orgLogoUrl": null }
```

**PWA Testing on Mobile:**
1. Open Safari/Chrome on iOS/Android
2. Navigate to https://staging-lodgra.vercel.app
3. Tap Share → Add to Home Screen (iOS) or Menu → Install App (Android)
4. Verify app icon appears on home screen
5. Tap app icon → verify login screen displays
6. Enter staging credentials → verify org data loads

**Edge Cases:**
- Non-existent email in staging → should return nulls
- Network offline → PWA cache should show cached login screen
- App reinstall → should start fresh at login screen
