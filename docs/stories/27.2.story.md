## 📋 Story

**As a** SEO specialist optimizing lodgra.io  
**I want** to implement on-page SEO optimizations across all pages  
**So that** pages rank higher for target keywords  

---

## 🎯 Acceptance Criteria

### AC1: Meta Tags Optimization
- [ ] Meta titles updated on all key pages (50-60 chars, keyword-first)
- [ ] Meta descriptions updated (150-160 chars, CTA-driven)
- [ ] **Success Metric:** 100% of pages have optimized title + description tags
- [ ] Primary keyword present in 80%+ of title tags
- [ ] Descriptions include compelling CTA (e.g., "Start free" / "Learn how")

### AC2: H1/H2 Hierarchy
- [ ] All pages have single H1 with primary keyword
- [ ] H2 headers structure content logically (3-5 per page)
- [ ] **Success Metric:** No duplicate H1 tags on any page
- [ ] Headers include target keywords naturally (no keyword stuffing)
- [ ] Header hierarchy: H1 > H2 > H3 (no skips)

### AC3: Internal Linking
- [ ] 3-5 internal links added per page (contextual)
- [ ] Links use descriptive anchor text (not "click here")
- [ ] **Success Metric:** Link structure creates 5+ topic clusters
- [ ] Orphan pages linked from main navigation or related content
- [ ] Total internal links increased by 50%+

### AC4: Image Optimization
- [ ] All images have descriptive alt text (keyword-relevant)
- [ ] **Success Metric:** 100% of images have alt text
- [ ] File names include keywords where relevant (example: "luxury-apartment-lisbon.jpg")
- [ ] Image compression: <100KB per image (lossy compression applied)
- [ ] Lazy loading enabled for below-fold images

### AC5: Schema Markup Verification
- [ ] LodgingBusiness schema verified (Story 26.2) — no errors in Rich Results Test
- [ ] LocalBusiness schema added (if applicable)
- [ ] Organization schema added to homepage
- [ ] **Success Metric:** Zero CRITICAL errors in Google Rich Results Test
- [ ] All schema tested with Google Rich Results Test tool

### AC6: URL Structure
- [ ] URLs are descriptive and keyword-inclusive (example: /properties/luxury-lisbon vs /prop/123)
- [ ] **Success Metric:** 100% of URLs use hyphens (not underscores)
- [ ] No duplicate content (canonicalization verified)
- [ ] Breadcrumb navigation added (schema + visible HTML)
