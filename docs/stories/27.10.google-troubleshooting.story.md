# Story 27.10 — Google Vacation Rentals Troubleshooting & Diagnostics

**Epic:** 27 — Google Vacation Rentals Integration  
**Status:** InProgress  
**Story ID:** 27.10  
**Created:** 2026-07-17  
**Type:** Backend / Diagnostic Feature  

---

## 📋 Story

**As a** property manager  
**I want** to see why a property is not indexed or showing in Google Vacation Rentals  
**So that** I can fix issues and get my properties visible on Google  

---

## 🎯 Acceptance Criteria

### AC1: Real-Time Validation Dashboard
- [x] Check property feed XML validity (schema validation)
- [x] Verify all required fields present (name, address, price, photos, rates)
- [x] Validate field formats (phone, email, dates, prices)
- [x] Check for common issues: missing photos, incomplete address, invalid currency
- [x] **Success Metric:** Validation completes <2s per property

### AC2: Google Indexation Status
- [x] Query Google Merchant Center API for indexation status per property
- [x] Display status: Indexed ✓, Pending, Error, Rejected
- [x] Show rejection reason (if applicable): "Missing price", "Invalid address", etc
- [x] Link to Google Merchant Center help articles
- [x] **Success Metric:** Sync status within 1h of Google update

### AC3: Historical Issue Tracking
- [x] Log all validation errors and resolution attempts
- [x] Show timeline: Issue → Fix Applied → Resolved (or Still Failing)
- [x] Alert if same issue persists for >7 days
- [x] **Success Metric:** 100% of issues tracked in audit log

### AC4: Auto-Fix Suggestions
- [x] Detect common issues: missing alt text, duplicate listings, invalid rates
- [x] Suggest fixes with one-click apply: "Add missing photo", "Update price"
- [x] Batch fixes: "Apply to all properties with this issue"
- [x] **Success Metric:** 80%+ of common issues fixable via UI

### AC5: Integration with Property Edit Flow
- [x] Show troubleshooting as inline warnings on property edit page
- [x] Real-time validation as user types (debounce 500ms)
- [x] Save automatically fixes issues (e.g., trim whitespace, format phone)
- [x] **Success Metric:** No validation errors after save

### AC6: Reporting & Export
- [x] Generate "Health Report": % of properties indexed, common issues, recommendations
- [x] Export diagnostic data: CSV or PDF
- [x] Schedule weekly report email (opt-in)
- [x] **Success Metric:** Report generates <10s for 100+ properties

## ⚠️ Risks

- Google API returns limited diagnostic info (may need reverse-engineering)
- Auto-fix could inadvertently change property data (need careful UX)
- Validation schema changes as Google updates (maintenance burden)
