# Story 27.10 — Google Vacation Rentals Troubleshooting & Diagnostics

**Epic:** 27 — Google Vacation Rentals Integration  
**Status:** Draft  
**Story ID:** 27.10  
**Created:** 2026-07-17  
**Type:** Backend / Diagnostic Feature  

---

## 📋 Story

**As a** property manager  
**I want** to see why a property is not indexed or showing in Google Vacation Rentals  
**So that** I can fix issues and get my properties visible on Google  

---

## 🎯 Acceptance Criteria

### AC1: Real-Time Validation Dashboard
- [ ] Check property feed XML validity (schema validation)
- [ ] Verify all required fields present (name, address, price, photos, rates)
- [ ] Validate field formats (phone, email, dates, prices)
- [ ] Check for common issues: missing photos, incomplete address, invalid currency
- [ ] **Success Metric:** Validation completes <2s per property

### AC2: Google Indexation Status
- [ ] Query Google Merchant Center API for indexation status per property
- [ ] Display status: Indexed ✓, Pending, Error, Rejected
- [ ] Show rejection reason (if applicable): "Missing price", "Invalid address", etc
- [ ] Link to Google Merchant Center help articles
- [ ] **Success Metric:** Sync status within 1h of Google update

### AC3: Historical Issue Tracking
- [ ] Log all validation errors and resolution attempts
- [ ] Show timeline: Issue → Fix Applied → Resolved (or Still Failing)
- [ ] Alert if same issue persists for >7 days
- [ ] **Success Metric:** 100% of issues tracked in audit log

### AC4: Auto-Fix Suggestions
- [ ] Detect common issues: missing alt text, duplicate listings, invalid rates
- [ ] Suggest fixes with one-click apply: "Add missing photo", "Update price"
- [ ] Batch fixes: "Apply to all properties with this issue"
- [ ] **Success Metric:** 80%+ of common issues fixable via UI

### AC5: Integration with Property Edit Flow
- [ ] Show troubleshooting as inline warnings on property edit page
- [ ] Real-time validation as user types (debounce 500ms)
- [ ] Save automatically fixes issues (e.g., trim whitespace, format phone)
- [ ] **Success Metric:** No validation errors after save

### AC6: Reporting & Export
- [ ] Generate "Health Report": % of properties indexed, common issues, recommendations
- [ ] Export diagnostic data: CSV or PDF
- [ ] Schedule weekly report email (opt-in)
- [ ] **Success Metric:** Report generates <10s for 100+ properties

- Auto-fix could inadvertently change property data (need careful UX)
- Validation schema changes as Google updates (maintenance burden)
