# Story 25.2: Admin — Gestão de Reviews por Propriedade (CRUD + Destaque)

**Epic:** 25 — OTA Reviews: Social Proof na Página de Reserva Direta

## Story

**Como** gestor de propriedades,
**Quero** adicionar, editar, eliminar e destacar reviews de OTAs no painel admin,
**Para que** possa curar o social proof que aparece na página pública de cada imóvel.

- [ ] **AC1** — Gestor pode adicionar uma review por propriedade: fonte OTA, nota (1–10), texto (opcional, max 500 chars), nome do hóspede, data
- [ ] **AC2** — Gestor pode editar qualquer campo de uma review existente
- [ ] **AC3** — Gestor pode eliminar uma review (com confirmação)
- [ ] **AC4** — Gestor pode marcar/desmarcar uma review como "destaque"
- [ ] **AC5** — Máximo 6 reviews em destaque por propriedade (validação no servidor)

- [ ] **AC8** — Isolamento multi-tenant: gestor só vê/edita reviews da sua organização
- [ ] **AC9** — Estados de loading e erro visíveis ao utilizador

### Padrão de Autenticação nas API Routes
[Source: src/app/api/cleaning/route.ts]

```typescript
import { NextRequest, NextResponse } from 'next/server'
import { createClient } from '@/lib/supabase/server'

export async function GET(request: NextRequest, { params }: { params: Promise<{ id: string }> }) {
  const supabase = await createClient()
  const { data: { user } } = await supabase.auth.getUser()
  if (!user) return NextResponse.json({ error: 'Unauthorized' }, { status: 401 })

  const { data: profile } = await supabase
    .from('user_profiles')
    .select('organization_id, role')
    .eq('id', user.id)
    .maybeSingle()

  if (!profile?.organization_id) {
    return NextResponse.json({ error: 'Forbidden' }, { status: 403 })
  }

  const { id: propertyId } = await params
  // ... lógica
}
```

Para POST/PUT/DELETE, verificar também `role`: `['admin', 'manager'].includes(profile.role)`

### Validação do Limite de Destaque (AC5)

No `PUT` handler, antes de marcar como featured:

```typescript
if (body.is_featured === true) {
  const { count } = await supabase
    .from('property_reviews')
    .select('id', { count: 'exact', head: true })
    .eq('property_id', propertyId)
    .eq('is_featured', true)
    .neq('id', reviewId)  // excluir a própria review

  if ((count ?? 0) >= 6) {
    return NextResponse.json(
      { error: 'Máximo de 6 reviews em destaque atingido.' },
      { status: 422 }
    )
  }
}
```
