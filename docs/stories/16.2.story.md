## Story

**Como** gestor de propriedades,
**Quero** preencher o número de adultos e crianças ao criar ou editar uma reserva,
**Para que** estes dados fiquem registados e disponíveis para operações.

- [x] **AC3** — Campos visíveis e editáveis no formulário de edição de reserva
- [x] **AC4** — Valores persistidos correctamente ao submeter o formulário
- [x] **AC5** — Página de detalhe de reserva mostra adultos e crianças
- [x] **AC6** — Validação: adultos ≥ 1, crianças ≥ 0
