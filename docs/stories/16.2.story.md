# Story 16.2: UI — campos adultos/crianças no formulário de reserva

**Epic:** 16 — Reservas: Dados de Hóspedes e PDF
**Status:** Ready for Dev
**Prioridade:** P2 — Depende de 16.1
**Tipo:** Frontend
**Agente:** @dev

---

## Story

**Como** gestor de propriedades,
**Quero** preencher o número de adultos e crianças ao criar ou editar uma reserva,
**Para que** estes dados fiquem registados e disponíveis para operações.

---

## Acceptance Criteria

- [ ] **AC1** — Campo "Adultos" (número inteiro ≥ 1) visível no formulário de criação
- [ ] **AC2** — Campo "Crianças (até 12 anos)" (número inteiro ≥ 0) visível no formulário
- [ ] **AC3** — Campos visíveis e editáveis no formulário de edição de reserva
- [ ] **AC4** — Valores persistidos correctamente ao submeter o formulário
- [ ] **AC5** — Página de detalhe de reserva mostra adultos e crianças
- [ ] **AC6** — Validação: adultos ≥ 1, crianças ≥ 0

---
