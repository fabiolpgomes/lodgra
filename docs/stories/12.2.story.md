## Story

**Para que** aceda a funcionalidades premium e Lodgra receba pagamentos mensais em BRL.

---

## Contexto de Negócio

Lodgra monetiza via subscriptions mensais. Cada organização escolhe um plano:
- **Starter:** R$59/mês (1 unidade, básico)
- **Professional:** R$89/mês + R$5/reserva (1-5 unidades)
- **Enterprise:** R$130/mês + 1% da receita (5+ unidades)

Inclui 14 dias de trial e billing na moeda local (BRL). Webhook processa eventos de atualização de subscrição.

### O que já existe
- Stripe customer criado na signup
- Tabela organizations com campos de billing

### O que falta
- Planos Stripe criados em Stripe Brasil
- Subscription endpoints (create, list, cancel, upgrade/downgrade)
- Webhook listeners para subscription events
- Dashboard UI (/billing) para ver/alterar plano
- Trial period logic (14 days)
- Past-due handling e email alerts

## Dev Notes

### Planos Stripe (Manual Setup)

```
Starter: R$59/month, 1 unit
Professional: R$89/month (base) + R$5 per booking, 1-5 units
Enterprise: R$130/month (base) + 1% revenue, 5+ units

Trial: 14 days (todos os planos)
```

### Subscription API Pattern

```typescript
// POST /api/billing/subscription
POST /api/billing/subscription?plan=professional

// GET /api/billing/subscription
GET /api/billing/subscription
→ { subscription_id, plan, status, current_period_end, trial_end, ... }

// PUT /api/billing/subscription
PUT /api/billing/subscription?plan=enterprise
→ { new_plan, proration_credits, next_billing_date }

// DELETE /api/billing/subscription
DELETE /api/billing/subscription?mode=now|end_of_period
```

### Webhook Events to Handle

```
customer.subscription.created
customer.subscription.updated (trial end, upgrade, downgrade)
customer.subscription.deleted
invoice.created
invoice.payment_succeeded (update subscription_status to 'active')
invoice.payment_failed (update subscription_status to 'past_due')
```

### Past-Due Logic

- payment_failed → subscription_status = 'past_due'
- Send email alert on day 0 and day 7
- Auto-cancel on day 30 if not resolved
- Grace period: 30 days
