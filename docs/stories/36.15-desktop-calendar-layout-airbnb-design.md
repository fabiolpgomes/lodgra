# Story 36.15: Desktop Calendar Layout (Airbnb Design Reference)

**Epic:** 36 (Dynamic Pricing + Calendar Redesign)  
**Story ID:** 36.15  
**Status:** Draft  
**Story Points:** 6  
**Priority:** HIGH  
**Scope:** Desktop layout for calendar with sidebar-based settings

---

## User Story

As a **Desktop Host**, I want **to see my calendar with a sidebar-based layout for settings and property list** so that **I can leverage the extra screen space for efficiency and quick access to all configurations**.

---

## Description

Extend Story 36.14 (mobile-first) to desktop (>768px) with:
- Left sidebar: Property list (vertical scrollable)
- Center: Detailed calendar with full month visible
- Right sidebar: Settings tabs (Preços | Descontos | Disponibilidade)
- 3-column layout optimized for large screens

**Why:** Desktop users benefit from seeing all information simultaneously. Sidebar layout reduces navigation, improves workflow efficiency.

---

## Acceptance Criteria

### Layout Structure (Desktop ≥1024px)
- [x] 3-column layout: Left sidebar (300px) | Center calendar (600px) | Right sidebar (300px)
- [x] Left sidebar: Property list (vertical scroll)
- [x] Center: Calendar grid (full month visible)
- [x] Right sidebar: Settings tabs
- [x] Gutters: 16-24px between columns
- [x] All columns scrollable independently

### Left Sidebar - Property List
- [x] Vertical list of properties (not horizontal cards like mobile)
- [x] Property item: Thumbnail (small, 80×80px) + name + type
- [x] Current property highlighted/selected state
- [x] Tap to switch property (updates calendar + settings)
- [x] Scrollable if 10+ properties
- [x] Search/filter input at top (optional)

### Center - Calendar View
- [x] Header: Property name | [📅 Month picker] [⚙️ Settings toggle]
- [x] Month/year selector for quick navigation
- [x] Calendar grid: Full month visible (7×6 grid max)
- [x] Day cells larger than mobile (accommodate more content)
- [x] Each day: Day number (large) + Price (bold) + Guest name (if booked)
- [x] Visual indicators:
  - Booked days: Light gray background + guest name
  - Weekend days: Subtle color difference
  - Today: Red circle on day number
  - Custom prices: Highlight or badge
- [x] Tap day → Quick edit modal (from Story 36.3)
- [x] Hover effects: Day cells highlight on mouse-over
- [x] Responsive: Adapts to screen width (1024px, 1440px, 1920px+)

### Right Sidebar - Settings Tabs
- [x] Tab navigation: Preços | Descontos | Disponibilidade
- [x] Tabs vertical or horizontal (based on design)
- [x] Tab content scrollable vertically (sidebar height)
- [x] Settings persist when switching properties
- [x] Save button at bottom of sidebar

### Desktop-Specific Features
- [x] Drag-and-drop to reschedule reservations (optional, enhanced feature)
- [x] Multi-select days for bulk operations (optional)
- [x] Tooltip on hover showing full guest name (if truncated)
- [x] Keyboard shortcuts for month navigation (← → arrows)

### Responsive Breakpoints
- [x] 768px - 1024px: Transitional (sidebar still visible but narrower)
- [x] 1024px - 1440px: Ideal 3-column layout
- [x] 1440px - 1920px: Expanded gutters, more whitespace
- [x] 1920px+: Extra columns or expanded sidebar widths

### Visual Design (Desktop)
- [x] Typography: Same as mobile but optimize for readability at distance
- [x] Spacing: Generous padding in sidebars (20-24px)
- [x] Calendar cells: Larger padding, clear visual hierarchy
- [x] Sidebar width: Adjust to screen size (300-400px on large screens)
- [x] Hover states: Cards and buttons highlight on mouse-over
- [x] Focus states: Keyboard navigation accessible

## Technical Notes

- Reuse components from Story 36.14 (mobile)
- Use CSS Grid or Flexbox for 3-column layout
- Breakpoint strategy: max-width media queries for responsive tiers
- Sidebar widths: Dynamic based on viewport (300px-400px)
- Calendar grid: Adjust cell padding/font size per breakpoint
- Focus management: Keyboard nav accessible for all interactive elements
