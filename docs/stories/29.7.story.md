## Endpoints

### GET /api/cleaning/tasks
Busca tarefas com filtros opcionais:
```
?status=pending&propertyId=...&cleanerId=...&startDate=2026-06-16&endDate=2026-06-17
```

### PATCH /api/cleaning/tasks
Atualiza status de uma tarefa:
```json
{
  "taskId": "uuid",
  "status": "in_progress|done|issue",
  "notes": "Optional notes"
}
```

## Onde está o número do cleaner?

**Resposta:** Tabela `user_profiles`, coluna `phone`

Fluxo de dados:
```
cleaning_tasks.cleaner_id (UUID)
            ↓
    user_profiles.id (UUID)
            ↓
    user_profiles.phone (STRING: "+5585988881111")
```

Quando crias uma tarefa:
```sql
INSERT INTO cleaning_tasks (
  cleaner_id,  -- ← references user_profiles.id
  ...
)

-- Para obter o phone:
SELECT u.phone
FROM user_profiles u
WHERE u.id = cleaning_tasks.cleaner_id;
```

**Na app:**
```typescript
// Já retornado no select() da API:
{
  "cleaner": {
    "full_name": "Maria",
    "phone": "+5585988881111",  // ← AQUI ESTÁ O NÚMERO
    "email": "maria@example.com"
  }
}
```

## Utilização (Para Gestor)

### Acessar Dashboard
```
https://www.lodgra.io/settings/cleaning
```

### Ver Tarefas
- Dashboard mostra todas as tarefas de hoje
- Cada card mostra: propriedade, cleaner (com telefone), status, progresso

### Contactar Cleaner
- Clica no number de telefone → abre WhatsApp/chamada

### Atualizar Status
1. Clica dropdown "Status"
2. Seleciona: Pendente, Em Progresso, Feita, Problema
3. Sistema salva automaticamente

### Filtrar Tarefas
- Por Data (date picker)
- Por Status (dropdown)
