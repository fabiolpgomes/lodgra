## Story

**Como** utilizador da plataforma ou visitante de uma página de propriedade,
**Quero** ver uma interface visualmente consistente com a identidade da marca Home Stay,
**Para que** a experiência transmita confiança e profissionalismo em todos os pontos de contacto.

---

## Contexto de Negócio

A plataforma usa Tailwind CSS com configuração padrão. Para suportar o redesign da página pública (Story 14.2) com uma identidade visual coesa, é necessário definir um sistema de design com:
- Paleta de cores da marca (primária, secundária, neutros, semânticas)
- Tipografia (família, escala, pesos)
- Espaçamento e radii consistentes
- Tokens CSS custom properties para uso em componentes

### Paleta Home Stay (proposta)

| Token | Valor | Uso |
|-------|-------|-----|
| `brand-50` a `brand-950` | Escala de azul-petróleo/verde-água | Cor primária |
| `accent-400` a `accent-600` | Coral/terracota suave | CTAs, destaques |
| `neutral-50` a `neutral-950` | Cinzas quentes | Texto, fundos |
| `success` | Verde | Disponível, confirmado |
| `warning` | Âmbar | Alertas, datas bloqueadas |
| `error` | Vermelho | Erros |

> **Nota para o dev:** A paleta exacta deve ser definida com o utilizador. As cores abaixo são um ponto de partida. Perguntar ao utilizador se tem brand guidelines ou preferências de cor antes de implementar.
