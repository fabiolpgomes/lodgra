**Type:** Backend / Reporting

## 📋 Story

**As a** property manager  
**I want** to export distribution metrics to CSV/PDF and schedule reports  
**So that** I can share insights with team/accountant and track trends over time

---

## 🎯 Acceptance Criteria

### AC1: CSV Export
- [ ] Export all platform metrics to CSV
- [ ] Include: date, platform, impressions, clicks, conversions, revenue, CTR%, conversion%
- [ ] Date range selector (7/30/90 days, custom)
- [ ] Generate in <5s
- [ ] **Success Metric:** CSV validates in Excel/Google Sheets

### AC2: PDF Report Generation
- [ ] Generate professional PDF with charts
- [ ] Include: summary table, market share pie chart, trends line chart
- [ ] Branded header (Lodgra logo)
- [ ] Date range in report title
- [ ] **Success Metric:** PDF renders without errors

### AC3: Scheduled Weekly Report
- [ ] User can enable weekly email report
- [ ] Sent every Monday 9 AM in user's timezone
- [ ] Includes: summary stats, market share, top platform
- [ ] Unsubscribe link included
- [ ] **Success Metric:** Email delivered successfully

### AC4: Scheduled Monthly Report
- [ ] User can enable monthly email report
- [ ] Sent on 1st of month 9 AM
- [ ] Includes: month summary, comparison to previous month, trends
- [ ] Unsubscribe link included
- [ ] **Success Metric:** Email delivered successfully

### AC5: Share Dashboard Link
- [ ] Generate read-only dashboard link (expires after 30 days)
- [ ] Share with team/accountant
- [ ] Link includes: property selection, date range
- [ ] No edit permissions for shared links
- [ ] **Success Metric:** Link accessible, read-only confirmed

### AC6: Financial Integration
- [ ] Export supports accounting software formats (CSV for Xero/QuickBooks)
- [ ] Include mapping to expense categories
- [ ] Revenue by channel for P&L reporting
- [ ] **Success Metric:** Imports correctly into Xero test account
