# Story 25.4: Página Pública — Carousel de Review Cards em Destaque

**Epic:** 25 — OTA Reviews: Social Proof na Página de Reserva Direta
**Status:** Done
**Prioridade:** P1 — Depende de Story 25.2 (is_featured) e 25.3 (padrão de integração)
**Tipo:** Frontend (Client Component + integração server)
**Agente:** @dev

---

## Story

**Como** visitante na página pública de uma propriedade,
**Quero** ver um carousel com as reviews em destaque,
**Para que** possa ler comentários reais de outros hóspedes e ganhar confiança para reservar.

---

## Acceptance Criteria

- [ ] **AC1** — Carousel visível em `/p/[slug]` apenas se existirem reviews com `is_featured = true`
- [ ] **AC2** — Máximo 6 cards no carousel (apenas reviews com `is_featured = true`)
- [ ] **AC3** — Cada card exibe: texto da review (truncado a 3 linhas com "Ver mais"), nome do hóspede, mês/ano da data, nome da fonte OTA, nota (1–10)
- [ ] **AC4** — "Ver mais" expande o texto completo inline (sem modal)
- [ ] **AC5** — Carousel com scroll horizontal em mobile (overflow-x: auto, snap)
- [ ] **AC6** — Em desktop: grid de 2 ou 3 colunas (sem scroll)
- [ ] **AC7** — Dados carregados server-side na página `/p/[slug]` (sem fetch client-side extra)
- [ ] **AC8** — Secção posicionada após `PropertyReviewScore` na página pública

## Dev Notes

### Query das reviews em destaque
[Source: src/app/p/[slug]/page.tsx — padrão já estabelecido em Story 25.3]

```typescript
const { data: featuredReviewsRaw } = await adminClient
  .from('property_reviews')
  .select('*')
  .eq('property_id', property.id)
  .eq('is_featured', true)
  .order('review_date', { ascending: false })
  .limit(6)

const featuredReviews = (featuredReviewsRaw ?? []) as PropertyReview[]
```

### Tipo `PropertyReview`

```typescript
export type PropertyReview = {

  rating: number
  review_text: string | null
  review_date: string       // 'YYYY-MM-DD'
  is_featured: boolean

  updated_at: string
}
```

### Formatação da data (mês/ano)

```typescript
function formatDate(dateStr: string): string {
  const d = new Date(dateStr + 'T00:00:00')
  return d.toLocaleDateString('pt-PT', { month: 'long', year: 'numeric' })
}
```
