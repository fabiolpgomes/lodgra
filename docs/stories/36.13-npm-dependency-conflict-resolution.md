## User Story

As a **DevOps Engineer**, I want **to resolve the npm install conflict between artillery and uuid packages** so that **CI/CD pipelines can run successfully and developers can install dependencies without errors**.

---

## Description

The project has a dependency conflict between artillery (testing framework) and uuid package versions that causes npm install to fail or hang. This blocks CI/CD setup and development environment bootstrap.

**Why:** Unresolved dependency conflicts prevent automated testing infrastructure setup and slow developer onboarding.
