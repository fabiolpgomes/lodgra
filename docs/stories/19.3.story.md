## Dev Notes

### Estrutura JSON-LD alvo

```json
{
  "@context": "https://schema.org",
  "@type": "VacationRental",
  "name": "...",
  "checkinTime": "15:00",
  "checkoutTime": "11:00",
  "makesOffer": [
    {
      "@type": "Offer",
      "name": "Preço por noite",
      "price": 120,
      "priceCurrency": "EUR",
      "unitCode": "DAY",
      "eligibleQuantity": {
        "@type": "QuantitativeValue",
        "minValue": 2,
        "unitText": "noites"
      }
    },
    {
      "@type": "Offer",
      "name": "Taxa de limpeza",
      "price": 50,
      "priceCurrency": "EUR",
      "description": "por estadia"
    }
  ]
}
```
