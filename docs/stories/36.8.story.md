## Dev Notes

### Database Schema

```sql
-- Create pricing_recommendations table
CREATE TABLE pricing_recommendations (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  property_id UUID NOT NULL REFERENCES properties(id) ON DELETE CASCADE,
  recommended_price DECIMAL(10, 2) NOT NULL,
  confidence DECIMAL(3, 2) NOT NULL, -- 0.00-1.00
  reason TEXT NOT NULL,
  market_analysis JSONB, -- { median_price, market_trend, competitor_avg }
  revenue_projection JSONB, -- { current_monthly, projected_monthly, difference }
  accepted BOOLEAN DEFAULT FALSE,
  accepted_at TIMESTAMP,
  rejected_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

-- Audit trigger
CREATE TRIGGER pricing_recommendations_audit
BEFORE UPDATE ON pricing_recommendations
FOR EACH ROW EXECUTE FUNCTION audit_trigger();
```

### API Endpoints

**GET /api/properties/:id/recommendations**
- Generates fresh recommendations for property
- Returns: `{ recommendations: [{ id, price, confidence, reason, market_analysis, revenue_projection }] }`
- Status codes: 200, 404, 401

**POST /api/properties/:id/recommendations/:recommendationId/accept**
- Marks recommendation as accepted
- Updates property's pricing (if desired)
- Returns: `{ accepted: true, timestamp }`
- Status codes: 200, 404, 401

### Implementation Modules

### Technical Approach

1. **Demand Pattern Detection**
   - Analyze last 12 months of price-history records
   - Group by month and day-of-week
   - Calculate average nightly rates and occupancy
   - Identify trend (increasing/decreasing/stable)

2. **Market Benchmarking**
   - Simple static dataset of regional medians
   - Match property by region + property_type
   - Adjust for property features (beds, baths, amenities)
   - Calculate percentile ranking

3. **Recommendation Generation**
   - Base: Average of detected optimal price + market median
   - Adjustment: Apply confidence weighting
   - Confidence: Based on data quality (sample size, variance)
   - Reasoning: "High demand in July (20% above average), market median is $X, recommendation is $Y"

4. **Revenue Projection**
   - Current: Average monthly revenue (last 3 months)
   - Projected: Simulate revenue with recommended price using occupancy patterns
   - Show dollar impact and percentage change
