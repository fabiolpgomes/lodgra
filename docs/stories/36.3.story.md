## Story

**Como** anfitrião de propriedade de alojamento local,
**Quero** ver e editar preços de alojamento para datas específicas através de um calendário interactivo,
**Para que** eu possa gerir dinamicamente as tarifas sem acesso directo ao banco de dados, com interface intuitiva e mobile-first.

---

## Contexto

A gestão de preços por data é essencial para anfitriões que precisam variar tarifas com base em sazonalidade, eventos locais e períodos de alta/baixa procura. Um calendário mensal permite:
- Visualizar o panorama completo de preços de um mês
- Distinguir preços base, fins de semana e sobreposições
- Seleccionar datas para editar preços individuais
- Suportar seleção de intervalo para operações em massa

O Story 36.1 fornece o endpoint `GET /api/properties/:id/daily-prices?month=YYYY-MM` para obter dados. O Story 36.2 fornece o hook `usePricingSettings` para acesso ao contexto global de preços.

## Dev Notes

### Selecção de Biblioteca

Consideradas opções:
- **react-big-calendar**: potente mas pesada (bundles ~40KB gzipped)
- **react-calendar**: lightweight (~8KB), mas requer customização extra
- **Custom calendar**: máximo controlo, melhor mobile UX, tempo ~3-4h

**Decisão:** Implementar componente customizado usando React hooks + Tailwind. Benefícios:
- Controlo total sobre rendering e interacções
- Otimizado para mobile (swipe nativo, sem deps pesadas)
- Estrutura compreensível para futuras modificações
- Bundle size mínimo

### Hook useCalendarMonth

interface DailyPrice {
  date: string;              // ISO 8601: YYYY-MM-DD
  basePrice: number;
  weekendPrice?: number;
  override?: number;        // null = sem override
}

function useCalendarMonth(propertyId: string, month: Date) {
  const [prices, setPrices] = useState<Map<string, DailyPrice>>()
  const [loading, setLoading] = useState(false)
  const [error, setError] = useState<Error>()

  // Fetch ao montar e quando mês muda
  useEffect(() => {
    const yearMonth = month.toISOString().slice(0, 7)  // YYYY-MM
    fetch(`/api/properties/${propertyId}/daily-prices?month=${yearMonth}`)
      .then(r => r.json())
      .then(data => {
        const map = new Map(data.map(p => [p.date, p]))
        setPrices(map)
      })

      .finally(() => setLoading(false))

  }

  return { prices, loading, error, setPrice, deletePrice }

function getPriceIndicatorColor(price: DailyPrice): string {
  if (price.override) return 'bg-red-100 border-red-300'      // Override destacado
  if (isWeekend(price.date) && price.weekendPrice) 
    return 'bg-amber-100 border-amber-300'                    // Fim de semana
  return 'bg-blue-50 border-blue-200'                         // Base
}

### Optimistic Updates

```typescript
const setPrice = async (date: string, price: number) => {
  // 1. Atualizar UI imediatamente
  setPrices(prev => new Map(prev).set(date, { ...prev.get(date), override: price }))
  
  // 2. POST para API
  try {
    const res = await fetch(`/api/properties/${propertyId}/daily-prices`, {
      method: 'POST',
      body: JSON.stringify({ date, price })
    })
    if (!res.ok) throw new Error('Failed to save price')
    // 3. Se sucesso, re-fetch para confirmar (opcional)
    await refetchPrices()
  } catch (err) {
    // 4. Rollback se erro
    setPrices(prev => new Map(prev).set(date, { ...prev.get(date), override: undefined }))
    showErrorToast('Failed to save price')
  }
}
```
