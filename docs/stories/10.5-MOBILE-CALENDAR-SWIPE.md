## Story

As a mobile user, I want to interact with the calendar using touch gestures (swipe left/right to navigate months) and see a mobile-optimized view (smaller fonts, compact layout, responsive event display) so that I can manage reservations efficiently on my phone without struggling with cramped layouts or keyboard navigation.

---

## Acceptance Criteria

### AC1: FullCalendar Mobile Layout
- [ ] Calendar displays in month view on all screen sizes
- [ ] Responsive font sizes: labels 6px mobile → 10px tablet → 12px desktop
- [ ] Day cells stack in grid on mobile (7 columns, responsive height)
- [ ] Event height scales: compact on mobile (20px) → normal on tablet (32px)
- [ ] No horizontal scrolling on 320px screens
- [ ] Calendar container responsive: `max-w-full` on mobile

### AC2: Touch Gesture Support
- [ ] Swipe left: Navigate to next month
- [ ] Swipe right: Navigate to previous month
- [ ] Swipe sensitivity: minimum 30px horizontal movement
- [ ] No gesture triggers on vertical scroll
- [ ] Gesture feedback: visual indication of swipe direction
- [ ] Desktop fallback: Previous/Next buttons still work

### AC3: Responsive Event Display
- [ ] Mobile (320px): dayMaxEvents = 1 (show "+X more")
- [ ] Tablet (640px+): dayMaxEvents = 2
- [ ] Desktop (1024px+): dayMaxEvents = 3
- [ ] Event titles truncate gracefully on mobile
- [ ] Event colors preserved across sizes
- [ ] Overflow indicator visible when events hidden

### AC4: Mobile-Optimized Interactions
- [ ] Tap date: Show day view or details sheet (not modal)
- [ ] Bottom sheet for day details (swipe up/down to open/close)
- [ ] Long-press event: Show options (edit, delete, details)
- [ ] No hover states on mobile (use touch-active instead)
- [ ] Touch targets 44px+ minimum

### AC5: Performance & Accessibility
- [ ] Calendar loads < 2s on 4G
- [ ] WCAG AA: Touch targets 44px+, color contrast 4.5:1
- [ ] Keyboard navigation still works (Tab, Arrow keys, Enter)
- [ ] Screen reader: All interactive elements labeled
- [ ] No layout shift when events load
