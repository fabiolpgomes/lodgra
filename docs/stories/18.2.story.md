# Story 18.2: Comodidades por cômodo com ícones

**Epic:** 18 — Propriedades: Comodidades, Quartos, Banheiros e Taxas
**Status:** Ready for Dev
**Prioridade:** P2 — Depende de 18.1
**Tipo:** Full Stack
**Agente:** @dev

---

## Story

**Como** gestor de propriedades,
**Quero** seleccionar as comodidades da propriedade a partir de uma lista organizada por cômodo,
**Para que** os hóspedes saibam exactamente o que a propriedade oferece.

---

## Acceptance Criteria

- [ ] **AC1** — Secção "Comodidades" no formulário de edição de propriedade
- [ ] **AC2** — Lista organizada por categoria: Geral, Sala, Quarto, Cozinha, Banheiro, Segurança
- [ ] **AC3** — Cada comodidade exibe ícone Lucide + nome
- [ ] **AC4** — Checkboxes para seleccionar/desseleccionar
- [ ] **AC5** — Secção "Destaques" — comodidades marcáveis como destaque
- [ ] **AC6** — Secção "Itens de Segurança" separada visualmente
- [ ] **AC7** — Selecções persistidas em `property_amenities`
- [ ] **AC8** — Comodidades visíveis no perfil da propriedade com ícones
