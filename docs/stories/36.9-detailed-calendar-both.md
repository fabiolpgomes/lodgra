# Story 36.9: Detailed Calendar (Mobile + Web)

**Epic:** 36 (Dynamic Pricing + Calendar Redesign)  
**Story ID:** 36.9  
**Status:** Ready for Review  
**Story Points:** 8  
**Priority:** HIGH  

---

## User Story

As a **Host**, I want **to see a detailed month calendar with prices and reservations** so that **I can manage my property's pricing and bookings**.

---

## Description

Build main calendar view (reuses grid from Story 36.3) showing daily prices and reservations. Available in both mobile (full-screen) and web (center column) layouts.

**Why:** Central view for pricing + reservation management (core host workflow).

---

## Acceptance Criteria

### Mobile View

- [x] Full-screen calendar
- [x] Header: [← Back] | Property name | [📅] [⚙️]
- [x] Calendar grid (7×4 or 7×5 per month)

### Web View

- [x] Centered calendar (sidebar left, config right)
- [x] Header: Property name | [📅] [⚙️]
- [x] Same grid as mobile

### Calendar Grid (Both Layouts)

- [x] Display 7×4 or 7×5 grid (days of month)
- [x] Each cell: Day number + Price (EUR)
- [x] Navigation: ◄ Previous month | Month/Year | Next month ►
- [x] Tap day → Edit price modal (from Story 36.3)

### Reservations Display

- [x] Show guest names across booked days
- [x] Example: "João (5-9)" spans across July 5-9
- [x] Multiple reservations: Stack or show count (+3 more)
- [x] Guest names clickable to view reservation details (future)

### Visual Indicators

- [x] Booked days: Gray/dark background
- [x] Weekend days: Subtle color difference
- [x] Today: Red circle on day number
- [x] Custom price days: Bold or underline

### Settings Access

- [x] Tap ⚙️ → Open config panel / modal
- [x] Tap 📅 → Month/Year picker (optional, for faster navigation)
