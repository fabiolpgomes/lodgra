# Story 36.9: Detailed Calendar (Mobile + Web)

**Epic:** 36 (Dynamic Pricing + Calendar Redesign)  
**Story ID:** 36.9  
**Status:** Draft  
**Story Points:** 8  
**Priority:** HIGH  

---

## User Story

As a **Host**, I want **to see a detailed month calendar with prices and reservations** so that **I can manage my property's pricing and bookings**.

---

## Description

Build main calendar view (reuses grid from Story 36.3) showing daily prices and reservations. Available in both mobile (full-screen) and web (center column) layouts.

**Why:** Central view for pricing + reservation management (core host workflow).

---

## Acceptance Criteria

### Mobile View

- [ ] Full-screen calendar
- [ ] Header: [← Back] | Property name | [📅] [⚙️]
- [ ] Calendar grid (7×4 or 7×5 per month)

### Web View

- [ ] Centered calendar (sidebar left, config right)
- [ ] Header: Property name | [📅] [⚙️]
- [ ] Same grid as mobile

### Calendar Grid (Both Layouts)

- [ ] Display 7×4 or 7×5 grid (days of month)
- [ ] Each cell: Day number + Price (EUR)
- [ ] Navigation: ◄ Previous month | Month/Year | Next month ►
- [ ] Tap day → Edit price modal (from Story 36.3)

### Reservations Display

- [ ] Show guest names across booked days
- [ ] Example: "João (5-9)" spans across July 5-9
- [ ] Multiple reservations: Stack or show count (+3 more)
- [ ] Guest names clickable to view reservation details (future)

### Visual Indicators

- [ ] Booked days: Gray/dark background
- [ ] Weekend days: Subtle color difference
- [ ] Today: Red circle on day number
- [ ] Custom price days: Bold or underline

### Settings Access

- [ ] Tap ⚙️ → Open config panel / modal
- [ ] Tap 📅 → Month/Year picker (optional, for faster navigation)

---

## Definition of Done

- [ ] Calendar displays correctly for all months
