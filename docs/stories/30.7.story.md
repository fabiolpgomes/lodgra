# Story 30.7 — Reservation Confirmation: Confirmação Automática por WhatsApp

**Epic:** 30 — WhatsApp/SMS Automated Operations  
**Status:** Ready  
**Prioridade:** P1

## Story

**Como** hóspede,  
**Quero** receber uma mensagem WhatsApp confirmando a minha reserva imediatamente após a confirmação,  
**Para que** tenha comprovativo do booking e detalhes da propriedade de forma rápida.

---

## Contexto de Negócio

A confirmação imediata por WhatsApp reduz fricção na experiência de booking e fornece ao hóspede confiança de que a reserva foi processada com sucesso.

---

## Critérios de Aceitação

- [x] **AC1** — Mensagem enviada automaticamente quando reserva muda de status para `confirmed`
- [x] **AC2** — Mensagem inclui obrigatoriamente: título da propriedade, endereço completo, nome do hóspede, datas de check-in/checkout, preço total
- [x] **AC3** — Requer telefone do hóspede para envio
- [x] **AC4** — Log de envio em `whatsapp_logs`
- [x] **AC5** — Evitar duplo envio: campo `confirmation_sent_at` em `reservations`
- [x] **AC6** — Gestor pode reenviar manualmente via botão na página de reserva (`POST /api/reservations/[id]/send-confirmation`)
- [x] **AC7** — Fallback para email se hóspede sem WhatsApp
- [x] **AC8** — Testes: trigger, timing, duplicate prevention, fallback (7+ testes)
