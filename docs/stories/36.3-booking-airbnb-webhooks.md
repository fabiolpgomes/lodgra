# Story 36.3: Implementar Webhooks de Booking & Airbnb para Sincronização Bidirecional

**Status**: Draft  
**Priority**: 🟡 MEDIUM (Enables real-time sync)

## 🎯 Objetivo

Implementar **webhooks de Booking.com e Airbnb** para:
1. Receber notificações de **cancelamentos em tempo real** (não apenas iCal delayed)
2. Atualizar status de reservas **automaticamente**
3. Eliminar **lag de sincronização** (iCal leva minutos/horas)
4. Preparar para **integração bidirecional** futura

---

## 🔴 Problema Atual

| Cenário | Tempo | Problema |
|---------|-------|----------|
| Guest cancela no Booking | T+0 | ✅ Cancela lá imediatamente |
| Lodgra atualiza | T+5-60min | ❌ Lag due to iCal polling |
| Proprietário vê | T+5-60min | ❌ Pode vender datas canceladas |

**Resultado**: Overbooking, frustração, cancelamento de última hora.

---

## ✅ Solução Proposta

### Fluxo Webhook

```
Booking.com:
  Guest cancela reserva
       ↓
  Booking envia POST /api/webhooks/booking
       ↓
  Lodgra recebe webhook
       ↓
  Atualiza reserva → status = "cancelled"
       ↓
  Proprietário vê imediatamente ✅
```

### Subtask 2: Booking.com Webhook
**Files**: `src/app/api/webhooks/booking/reservation/route.ts`

**Eventos Suportados**:
```typescript
// booking_confirmed → status = "confirmed"
// reservation_changed → update check_in/check_out/guests
// reservation_cancelled → status = "cancelled"
// reservation_completed → status = "completed"

export async function POST(request: NextRequest) {
  const signature = request.headers.get('x-booking-signature')
  const event = await request.json()
  
  // 1. Validar assinatura (HMAC-SHA256)
  if (!validateBookingSignature(event, signature)) {
    return NextResponse.json({ error: 'Invalid signature' }, { status: 403 })
  }
  
  // 2. Buscar reserva pelo booking_reference
  const reservation = await supabase
    .from('reservations')
    .select('*')
    .eq('booking_reference', event.reservation.id)
    .single()
  
  // 3. Update baseado no evento
  const updates = mapBookingEventToUpdate(event)
  await supabase
    .from('reservations')
    .update(updates)
    .eq('id', reservation.id)
  
  // 4. Log para auditoria
  await logWebhookEvent('booking', event.id, updates)
  
  return NextResponse.json({ success: true })
}
```

**Acceptance**:
- [ ] Assinatura HMAC validada
- [ ] Todos os 4 eventos processados
- [ ] Reserva encontrada pelo booking_reference
- [ ] Status atualizado corretamente
- [ ] Log criado

```typescript
export async function POST(request: NextRequest) {
  const signature = request.headers.get('x-airbnb-signature')

  // 3. Update
  const updates = mapAirbnbEventToUpdate(event)
  await supabase
    .from('reservations')
    .update(updates)
    .eq('id', reservation.id)
  
  return NextResponse.json({ success: true })
}
```

```typescript
export function mapBookingEventToUpdate(event: any) {
  return {
    status: eventStatusMap[event.event_type],
    check_in: event.reservation.check_in,
    check_out: event.reservation.check_out,
    number_of_guests: event.reservation.guests,
    guest_phone: event.reservation.phone,
    updated_at: new Date().toISOString(),
    webhook_synced_at: new Date().toISOString()
  }
}

export function mapAirbnbEventToUpdate(event: any) {
  return {
    status: airbnbStatusMap[event.event_type],
    // Airbnb data disponível no payload
    updated_at: new Date().toISOString(),
    webhook_synced_at: new Date().toISOString()
  }
}
```
