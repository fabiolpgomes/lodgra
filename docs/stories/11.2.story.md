## Story

**Para que** eu compreenda como os meus dados são tratados conforme a legislação do meu país.

## Contexto de Negócio

Existem páginas de privacy e terms mas são **estáticas e só em PT**. Para compliance:
1. Devem ser traduzidas (PT, PT-BR, EN-US)
2. Devem conter cláusulas específicas por região (RGPD para PT, LGPD para BR, CCPA para US)
3. Devem ser versionadas (data da última actualização visível)
4. Utilizadores devem aceitar a versão actual no signup
