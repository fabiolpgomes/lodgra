## 🔧 Dev Notes

### How to Get Google Merchant API Access

**Step 1: Create Google Cloud Project**
```bash
# Go to: https://console.cloud.google.com
# Create new project: "Lodgra Google Sync"
# Enable APIs:
#   - Google Merchant API
#   - Google Merchant Center API
```

**Step 2: Create Service Account (Recommended)**
```bash
# In Google Cloud Console:
# 1. Go to "Service Accounts" 
# 2. Create new service account: lodgra-merchant-sync
# 3. Grant roles: "Merchant API Editor" or "Merchant Center Editor"
# 4. Create JSON key file
# 5. Download and add to .env.local:
#    GOOGLE_SERVICE_ACCOUNT_EMAIL=...@....iam.gserviceaccount.com
#    GOOGLE_SERVICE_ACCOUNT_KEY_ID=...
#    GOOGLE_SERVICE_ACCOUNT_PRIVATE_KEY=-----BEGIN PRIVATE KEY-----\n...
#    GOOGLE_MERCHANT_CENTER_ID=123456789
```

**Step 3: Alternative - OAuth2 Flow (if needed for user auth)**
```bash
# Use Google OAuth2 library
# Redirect user to Google login
# Exchange code for access token
# Store token for API calls
```

### API Endpoints

**Get Product Status:**
```
GET https://merchantapi.googleapis.com/merchantapi/reports_service_v1beta/accounts/{merchantId}/productStatus?pageSize=100
Headers: Authorization: Bearer {access_token}
```

**Response Example:**
```json
{
  "products": [
    {
      "name": "accounts/123/products/goog_abc123",
      "resourceStatus": {
        "destination": "Shopping",
        "status": "ACTIVE"
      },
      "googleExpirationDate": "2025-12-31"
    }
  ]
}
```

### Rate Limiting

- Quota: 500 requests per minute per project
- Implement exponential backoff (1s → 2s → 4s → 8s)
- Cache responses for 1 hour locally

1. Create `GoogleMerchantClient` class (lib/google/merchant-client.ts)
2. Create sync worker (lib/workers/google-merchant-sync.ts)
3. Setup database schema (migrations)
4. Create cron job or scheduled function
5. Update dashboard to use sync data
6. Add manual sync button with loading state

```typescript
const status = await googleClient.getProductStatus(propertyId)

  property_id: propertyId,

```

```typescript
// api/cron/google-merchant-sync.ts
export async function GET(req: Request) {
  const authHeader = req.headers.get('authorization')
  if (authHeader !== `Bearer ${process.env.CRON_SECRET}`) {
    return new Response('Unauthorized', { status: 401 })
  }
  
  // Run sync job
  await syncGoogleMerchantStatus()
  return new Response('Sync completed', { status: 200 })
}
```
