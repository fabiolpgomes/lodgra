# Keyword Research Template & Instructions

## 7 Primary Keywords Research

For each keyword, complete this template:

### Keyword Template

```
**Keyword:** [keyword here]
**Monthly Volume:** [number] searches/month
**Difficulty:** [1-100 scale, 30 is target]
**Intent:** [informational / commercial / transactional]
**CPC:** [cost per click if available]

**Top 3 Ranking Pages:**
1. [URL] - [title]
2. [URL] - [title]
3. [URL] - [title]

**Notes:** [observations about competition, opportunity]
```

### Instructions

1. **Where to research:**
   - Free: Google Keyword Planner, Google Autocomplete, Related Searches
   - Paid: Ahrefs, SEMrush, Ubersuggest (trial available)

2. **Data to collect per keyword:**
   - Monthly search volume (≥100 minimum)
   - Keyword difficulty (≤30 preferred)
   - Search intent (what users want: info, products, local?)
   - Top 3 ranking pages (analyze what ranks)
   - CPC if available (indicates commercial value)

3. **Export format:** CSV with columns:
   ```
   keyword,volume,difficulty,intent,cpc,top_page_1,top_page_2,top_page_3,priority_score
   ```

4. **Priority score calculation:**
   ```
   Priority = (Volume × Intent_Weight) / Difficulty
   - Informational: Weight 1x
   - Commercial: Weight 2x
   - Transactional: Weight 3x
   ```
