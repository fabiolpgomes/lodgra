# LODGRA Precision Design System

## Core Concept
A high-performance interface that bridges the gap between automotive engineering excellence and data infrastructure. The aesthetic is "Clinical Performance" — ultra-clean, high-contrast, and surgically precise.

## Visual Language
- **Precision over Comfort:** Zero border-radius on all elements to emphasize structural integrity and technical rigor.
- **Deep Contrast:** Using pure white (#ffffff) backgrounds with Navy Blue (#1E3A8A) typography for maximum legibility and a corporate-technical feel.
- **Strategic Accents:** 
  - **Lamborghini Gold (#ffc000):** Reserved exclusively for primary CTAs and critical performance indicators.
  - **Emerald Green (#059669):** Used for technical specs, success states, and secondary visual accents.

## Typography
- **Primary Font:** Hanken Grotesk.
- **Display Style:** Heavy weights (800-900), tight tracking, and frequent use of uppercase for headers to evoke the feel of high-end automotive branding.
