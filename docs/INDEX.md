# ÍNDICE DO PROJETO HOME STAY

## Projeto
- **Nome**: Home Stay
- **Objetivo**: Sistema de gestão de alojamentos multi-plataforma (Airbnb + Booking)
- **Stack**: Next.js + Supabase + Vercel
- **Escala MVP**: 10 propriedades
- **Foco**: UX rápido + sincronização robusta
